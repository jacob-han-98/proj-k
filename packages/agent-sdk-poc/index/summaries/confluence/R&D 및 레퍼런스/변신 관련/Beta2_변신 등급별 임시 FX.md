## 한 줄 설명
Project K의 변신 시스템에서 등급별(영웅, 전설, 신화) FX 표현 방향과 생산 스펙을 정의한 기획 문서. 높은 등급의 변신에 만족감을 제공하기 위해 등급별 고유 컬러 광원과 파티클/오라를 단계적으로 추가하는 방식을 제시한다.

## 핵심 용어
- 변신 등급 (영웅, 전설, 신화)
- 광원 FX
- 고유 컬러 (보라색, 노란색, 빨강색)
- 파티클
- 오라
- 캐릭터 뒷 레이어
- 장판형 FX
- 날개형 글로우
- 생산 리소스 규격화
- 레퍼런스 (로드나인, 레이븐2, 뱀피르, 오딘)
- 빛무리
- 투명도 조절
- 문양
- 장신구 부유
- 번개 분출 FX
- 디테일 상향

## 숫자/상수/공식
- 등급 별 광원: 3종 (영웅, 전설, 신화 각 1종)
- 고유 컬러 파티클: 1종
- 고유 컬러 오라: 1종
- 신화용 추가 FX: 1종 (Beta2에서는 미제작)
