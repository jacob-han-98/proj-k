## 한 줄 설명
Project K의 변신 시스템에서 등급별(영웅, 전설, 신화) FX 컨셉, 제작 리소스, 개발 요청사항을 정의하는 문서.

## 핵심 용어
- 변신 등급 (영웅, 전설, 신화)
- FX (Effect)
- 변신 창 (UI)
- 인게임 FX
- 사슴 뿔 문양
- 아우라
- 파티클
- 글로우
- 리프 (세계관 내 에너지 요소)
- 메시
- 광원
- 하이폴 (High Polygon)
- 로우퀄 (Low Quality)
- 소켓 (Socket)
- FX_Grade
- FX_GradeHigh
- 뱀피르 (레퍼런스 변신)
- 정령 (Pet)
- 도깨비 (Companion)
- MetamorphGradeInfo 시트
- PetGradeInfo 시트

## 숫자/상수/공식
- 변신 창 / 인게임 기준 6개 리소스
- 변신창 UI 및 인게임 최대 줌 추가 연출 2개 리소스 (영웅 제외)
- 영웅 ↔ 전설 디테일 차이 > 전설 ↔ 신화 디테일 차이
- 사슴 뿔 갈라짐: 영웅(1~2갈래) → 전설(조금 더 갈라짐) → 신화(뿔이 원형 문양 뚫고 나옴)

## 참조 시스템

- Pet.xlsx (PetGradeInfo 시트)
- Companion 시트 (도깨비 신규 시트)

- 제작 필요 리소스 정리
- 개발 요청
- 등급 FX 구현
- 신규 소켓 2종 (FX_Grade, FX_GradeHigh)
- 레퍼런스 - 뱀피르 FX
