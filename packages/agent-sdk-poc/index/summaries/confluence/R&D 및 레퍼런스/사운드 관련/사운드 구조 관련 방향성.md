## 한 줄 설명
Project K의 MMORPG 사운드 시스템 구현 방향성을 정의하며, 사운드 분류, 우선순위, 클래스별 구현 방식, 항목별 상세 작업 가이드를 제시한다.

## 핵심 용어
- MMORPG 사운드 분류
- 내 PC 관련
- PvP/RvR 관련
- 강적 (PvE)
- 위험 알림/경고
- BGM + 환경
- 로그인/타이틀
- 보이스
- UI
- 사운드 중첩
- 우선순위 정렬
- 오디오 존/볼륨
- 사운드 클래스
- BGM
- Ambience
- SFX (사운드 이펙트)
- Voice
- Cinematic
- UE5 블루프린트 Modulator
- Foley
- Footstep
- 히트 이펙트
- 크리티컬

## 숫자/상수/공식
- 사운드 우선순위: 1순위 강적PvE, 2순위 내 PC 관련, 3순위 위험 알림/경고, 4순위 PvP/RvR 관련, 5순위 다른 PC 관련, 6순위 UI, 7순위 일반 몬스터, 8순위 보이스, 9순위 BGM/환경
- 이동 사운드(발자국): 피치 0.9~1.1, 볼륨 0.8~1.0
- 평타: 피치 0.9~1.0, 볼륨 0.8~1.0
- 강력한 스킬: 피치 0.95~1.05, 볼륨 0.95~1.0
- UI 창 호출 시 배경 사운드 볼륨 50% 감소
