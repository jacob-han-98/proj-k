## 한 줄 설명
Project K 개발 중 특정 맵 테스트, 클라이언트 독립실행, 리소스 최적화, 빌더 명령어 사용법을 정리한 테스트 가이드.

## 핵심 용어
- PKSingleGameMode
- 월드 세팅
- 게임모드 오버라이드
- 싱글플레이
- 서버 기반 작동
- 클라이언트 독립실행
- Default Maps
- Lobby
- 퍼포스
- ShowPkActorData
- EnableLandDebug
- TeleportTest
- servercommand
- SpawnMonster
- addbonus
- MoveSpeedRate
- TeleportLand
- show collision

## 숫자/상수/공식
- ShowPkActorData: 1(보여줌), 2(안보여줌)
- EnableLandDebug: 1(보여줌), 2(안보여줌)
- TeleportTest: 0(마을), 1(알파1 기준 가장 먼 위치), 2(마을 항구쪽)
- addbonus MoveSpeedRate 예시: 5000 (누적 가능, 음수로 원복)

- 특정 맵 로딩 후 싱글 플레이 테스트
- 클라이언트 독립실행시 서버 기반 동일 작동
- 백그라운드 컴터 리소스 최대 활용
- 클라 빌더명령어
