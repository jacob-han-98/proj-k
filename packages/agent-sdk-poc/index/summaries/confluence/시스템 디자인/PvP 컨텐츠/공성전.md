## 한 줄 설명

## 숫자/상수/공식
- 30~40분: 공성전 소요 시간
- 2주: 공성전 반복 주기
- 70%: 성물 각인 시 이동 속도 감소율
- 약 3분: 성물 각인 타이머
- 30분: 아무도 성물을 각인하지 않을 경우 수성 측 승리 판정 시간
- 30초 이상: 성물 각인 소요 시간 (최소값, 테스트 중)
