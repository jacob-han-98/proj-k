## 숫자/상수/공식
- 서버 이동권 1장 = 기본 1시간 이용 시간
- 최대 연장 시간: 3~5시간 예상
- 하루 최대 서버 이동 플레이 시간: 3~5시간
- 게임 종료 후 재접속 보장 시간: 10분 이내
- 복귀 캐스팅 타임: 약 10초
- 보너스 보스 HP: 매우 높음 (공/방 스탯 대비)
- 보너스 보스 제한 시간: 초과 시 디스폰
