## 한 줄 설명
여러 서버의 캐릭터들이 한시적으로 한 서버에 모여 보너스 보스를 공략하는 서버 이동 컨텐츠로, 필드 보스를 가장 먼저 처치한 서버에 추가 보스가 스폰되고 타 서버 유저들이 포탈을 통해 침공하여 보상을 경쟁하는 시스템.

## 숫자/상수/공식
- 결계 반지름: 15m (AreaRadius)
- 포탈 스폰 범위: 30cm (AttackServerPortalAreaRadius)
- 결계 밖 강제 귀환 대기 시간: 10초 (AttackServerAreaOutTime, 입력값 10000)
- 귀환 프로그레스 시간: 3초 (ReturnDurationTime, 입력값 3000)
- 스폰 확률: SpawnProb (확률 기반 스폰)
- 스폰 딜레이 시간: SpawnDelayTime (스폰 대기 상태 지속 시간)
- 스폰 유지 시간: SpawnHoldTime (보너스 보스 유지 시간)
- 클리어 딜레이 시간: ClearDelayTime (보스 디스폰 후 귀환까지 시간)
- 동일 서버 그룹 내 최대 1개 서버만 보너스 보스 스폰 가능
