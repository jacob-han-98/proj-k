## 한 줄 설명
타 서버로 이동하여 일정 시간 동안 PvPvE를 진행하는 컨텐츠로, 입장권 기반의 상시 이용 가능한 서버 이동 컨텐츠이다.

## 핵심 용어
- 서버 탐험
- 서버 침공
- 로컬 서버 캐릭터 (원주민)
- 타 서버 캐릭터 (이방인)
- 차원 이동 관리인
- 밀수 상인
- 경비병
- 코인 (Coin, 구 Kore)
- 차원석
- 입장권 (ExploreServerTicket)
- PvPvE
- 은신 상태
- 파티 규칙
- 네임 플레이트
- 월드맵
- 필드 선택
- 귀환 (수동/자동)
- 로그아웃 처리
- 재접속 타임아웃

## 숫자/상수/공식
- 일일 이용 가능 시간: 180분 (ExploreServerDailyTime)
- 나가기 버튼 귀환 프로그레스 시간: 3초 (ReturnDurationTime = 3000, 입력값 1000 = 1초)
- 로그아웃 시 남은 시간: 10초
- 코인 획득량: 몬스터당 고정 1개
- NativeKoreBaseRatio 공식: NativeKoreBaseRatio + ((몬스터 레벨 - 캐릭터 레벨) * NativeKoreAdjustRatio)
- StrangerKoreBaseRatio 공식: StrangerKoreBaseRatio + ((몬스터 레벨 - 캐릭터 레벨) * StrangerKoreAdjustRatio)
- 입력값 10,000 = 100% (비율 표기)
- 서버 포화 상태: 이방인 수 총량 0%
- 코인 상태 표기: 충분(100% 미만), 보통(70% 미만), 부족(40% 미만), 소진(0%)
- 이방인 수 표기: 여유(100% 미만), 보통(70% 미만), 혼잡(40% 미만), 포화(0%)
