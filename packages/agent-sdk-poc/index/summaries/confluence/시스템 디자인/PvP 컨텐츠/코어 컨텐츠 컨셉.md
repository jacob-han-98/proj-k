## 한 줄 설명
게임 플레이를 통해 캐시 재화로 환전 가능한 특수 재화 "코어"와 이를 획득하기 위한 PvPvE 지향 코어 컨텐츠들(필드 사냥, 필드 보스, 인터 서버 전장 등)을 정의하는 문서.

## 숫자/상수/공식
- 코어 드랍 한도: 서버 당 정해진 수량 제한
- 통화량 제한 해제 주기: 주간 단위
- 서버 잔여 코어 수량 표기: 4단계 (많음/보통/적음/매우 적음)
- 서버 침공 빈도 표기: 4단계 (많음/보통/적음/없음)
- 대시보드 갱신 주기: 1시간 단위
- 출입 오브젝트 사용 시간: 10~30초
- 코어존 게임 종료 후 증발 시간: 1시간
