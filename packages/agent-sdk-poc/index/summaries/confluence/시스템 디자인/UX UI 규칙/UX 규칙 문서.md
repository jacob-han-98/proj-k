## 한 줄 설명
Project K의 UX 설계 전반에 적용되는 규칙, 컴포넌트, 상호작용 기준을 정의한 상위 페이지로, 하위 문서들의 목차 및 내용 개요를 제시한다.

## 핵심 용어
- HUD 레이어
- UI LayerType
- HUD 기능적 레이아웃
- 해상도 대응
- 세이프 존 가이드라인
- 계층 구조(Hierarchy)
- 전환 UI FLOW
- 팝업 및 모달
- UI 딤드
- 제스처
- 등급 표현
- 버튼 UI
- 아이템 위젯
- 다이얼로그 팝업창
- 수량 슬라이더
- 리스트
- 폰트
- 알림 저장 규칙
- 시스템 메시지
- BM 상점 이동 FLOW
- Transition Duration
- Emotional Cue
- Spatial Logic
- UI 매니저

## 숫자/상수/공식
- Transition Duration: 0.3~0.8초 이내

## 참조 시스템
- UI 제작 규칙 및 팁
- UI LayerType 규칙
- 해상도 대응 계획
- 레드닷 시스템

## 주요 섹션
- 규칙 (HUD 레이어, 기능적 레이아웃, 해상도 대응, 세이프 존, 계층 구조, 전환 UI FLOW, 팝업 및 모달, UI 딤드, 제스처, 알림 저장, 시스템 메시지, BM 상점 이동)
- UI 컴포넌트 (버튼, 아이템 위젯, 다이얼로그, 수량 슬라이더, 리스트, 폰트)
