## 숫자/상수/공식
- 카운트다운: 24시간 이상 = n일, 24시간 미만 = HH:MM:SS, 1시간 미만 = MM:SS, 1분 미만 = 1분 미만 or 00초
- 누적 시간 기록용: HH:MM:SS (시:분:초)
- 누적 시간 보상용: 00시/60분 또는 게이지 바
- 유효 기간 장기 (3일 이상): YYYY.MM.DD HH:mm 만료
- 유효 기간 단기 (3일 미만 ~ 1일 이상): N일 N시간 남음
- 유효 기간 긴박 (24시간 미만): HH:mm:ss
- 보상 아이템 주요 보상 3~4개 노출 후 더보기 처리
- 숫자 단위 1만 미만: 전체 표기 (예: 8,500)
- 숫자 단위 1만 이상 ~ 1억 미만: '만' 단위 (예: 1,520만)
- 숫자 단위 1억 이상: 정밀형 1억 2,500만 또는 간소형 1.25억
- K (Kilo) = 1,000
- M (Mega) = 1,000,000
- B (Billion) = 1,000,000,000
- T (Trillion) = 1,000,000,000,000
