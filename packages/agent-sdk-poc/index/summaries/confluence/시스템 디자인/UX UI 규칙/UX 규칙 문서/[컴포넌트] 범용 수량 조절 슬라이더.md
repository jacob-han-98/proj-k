## 한 줄 설명
아이템 제작, 거래소, 가방 파기, 창고 입/출고 등에서 사용되는 수량 조절 기능을 범용 컴포넌트로 정의하며, 모바일(터치)과 PC(마우스/키보드) 플랫폼별 최적화된 조작 방식을 제시한다.
