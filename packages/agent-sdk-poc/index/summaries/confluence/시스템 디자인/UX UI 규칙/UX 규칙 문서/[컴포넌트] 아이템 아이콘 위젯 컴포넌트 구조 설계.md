## 한 줄 설명
인벤토리, 창고, 상점, 제작 등 모든 UI에서 공통으로 사용되는 아이템 슬롯 위젯의 구조, 데이터 바인딩 규칙, 정보 표기 방식을 정의하는 컴포넌트 설계 문서.

## 숫자/상수/공식
- 등급: 일반, 고급, 희귀, 영웅, 전설, 신화
- 보유수량 99 이상일 경우: +99로 표현
- 최대 소지량 이상: 999+
- Depth 0 (최하단) ~ Depth 3 (최상단)
- 우측 하단: 보유 수량, 강화/세공 수치
- 우측 상단: 아이템 티어
- 좌측 상단: 장착 표시
