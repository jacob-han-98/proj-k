## 한 줄 설명
내비게이션형 팝업 시스템을 통해 UI 계층 구조를 스택 기반으로 관리하고, 사용자가 뒤로가기로 이전 상태를 복원할 수 있도록 설계한 문서.

## 핵심 용어
- Navigation Stack Popup
- 뒤로가기 버튼
- 계층 구조 (Depth)
- Window / Sub Window
- 히스토리 스택 (History Stack)
- 시스템 팝업 (논 스택)
- 메인 윈도우
- 콘텐츠 윈도우
- 툴팁 및 서브 정보
- 휘발성 저장
- 클라이언트 메모리 저장
- 스택 제한
- 상태 저장
- 포커스 연출
- 전체 닫기 (X)
- 탭 상태 (Tab State)
- 스크롤 위치 (Scroll State)
- 선택 항목 (Selection)
- 정렬 필터 (Filter)

## 숫자/상수/공식
- 최대 스택 개수: 5~10개 (초과 시 최초 저장 Window부터 삭제)
- 최대 Depth: 10 (초과 시 가장 오래된 Stack [1] 삭제)
- 장시간 미조작 기준: 10분 이상 (강제 메인 탭 초기화)
