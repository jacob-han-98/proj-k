## 한 줄 설명
게임 내 아이템, 콘텐츠, 스킬 등의 가치 위계를 색상과 시각 연출로 구분하는 6단계 등급 체계 및 표시 규칙을 정의한 문서.

## 핵심 용어
- 등급 (Grade)
- 일반 (Common)
- 고급 (Uncommon)
- 희귀 (Rare)
- 영웅 (Unique)
- 전설 (Legendary)
- 신화 (Myth)
- 아이콘 BG 컬러
- 희소성 시각화
- 인벤토리
- 장비 아이템
- 변신/크리처
- 스킬 (액티브/패시브)
- 아이템 툴팁
- 카드 뒤집기 연출
- 상자 개봉 연출
- 정렬 우선순위

## 숫자/상수/공식
- 총 6가지 등급 존재
- 영웅 등급: 1단계 배경 연출
- 전설 등급: 2단계 배경 연출
- 신화 등급: 최고 등급 단계 배경 연출

## 참조 시스템
- 장비 아이템 발주 계획

## 주요 섹션
- 문서 개요
- 목표
- 등급 가이드
- 등급 별 정의 및 시각적 가이드
- 상황 별 사용 등급 가이드
- 장비 아이템 등급 표시 방식
- 아이템 (일반/재료) 등급 표시 방식
- 변신/크리처 등급 표시 방식
- 스킬 (액티브/패시브) 등급 표시 방식
- 아이템 툴팁 등급

- 등급 BG 희소성 시각화
- 획득 시 경험 연출
