## 한 줄 설명
다양한 모바일 디바이스의 노치, 카메라, 홈 바 등 하드웨어 영역을 회피하여 UI 요소가 안전하게 배치되는 세이프존(Safe Area)의 정의 및 적용 규칙을 명시한 문서.

## 핵심 용어
- 세이프존 (Safe Area)
- 세이프존 오프셋 (Offset)
- 노치 (Notch)
- 홀 펀치 (Hole Punch)
- 동적 아일랜드 (Dynamic Island)
- 홈 바 (Home Bar)
- 앵커 (Anchor)
- Full Screen UI
- HUD (Heads-Up Display)
- 고정 UI
- 외곽 UI
- 캔버스 스케일링
- Width/Height Match
- 레터박스 (Letter Box)
- 펀치홀 카메라
- 폴더블 디바이스
- 기준 해상도
- 종횡비 (Aspect Ratio)

## 숫자/상수/공식
- Left (좌): 100 px
- Right (우): 100 px
- Top (상): 32 px
- Bottom (하): 32 px
- 기준 해상도: 1920x1080 (FHD, 16:9)
- 표준 해상도: 1280x720(HD), 1600x900, 1920x1080(FHD)
- 긴 화면 비율: 19.5:9 ~ 21.5:9 (예: 2400x1080, 3120x1440)
- 태블릿 비율: 4:3 또는 5:6 (예: 2048x1536, 2208x1768)
- 가변 적용 기준: 기기 종횡비 1.5 미만일 경우 좌우 여백 80px 자동 축소
