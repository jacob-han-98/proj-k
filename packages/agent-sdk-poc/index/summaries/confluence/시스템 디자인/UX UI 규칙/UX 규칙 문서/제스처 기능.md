## 한 줄 설명
Project K 모바일 게임의 모바일/PC 플랫폼별 제스처 입력 매핑 및 스와이프 기능의 상세 구현 규칙을 정의한 문서.

## 핵심 용어
- 제스처 기능
- 탭 (Tap)
- 더블 탭 (Double Tap)
- 롱 프레스 (Long Press)
- 드래그 (Drag)
- 스와이프 / 플릭 (Swipe/Flick)
- 패닝 (Panning)
- 핀치 (Pinch & Spread)
- 회전 (Rotate)
- 클릭 (Click)
- 더블 클릭 (Double Click)
- 우클릭 (Right Click)
- 드래그 앤 드롭 (Drag & Drop)
- 스크롤 (Scroll)
- 확대/축소 (Zoom In/Out)
- 터치 시작 지점
- 드래그 속도 감도
- 최소 인식 거리
- 유효 스와이프 영역
- 전환 불가 조건

## 숫자/상수/공식
- 터치 시작 영역: 화면 좌/우측 가장자리로부터 20~30% (약 25%) 이내
- 드래그 이동 거리: 화면 폭의 70~80% (약 25%) 이상
- 최소 전환 거리: 화면 폭의 70% 이상
