## 한 줄 설명
Project K의 장비 드랍률, 제작식, 재료 수급 체계를 서버 인원 기반으로 정의하고, IBT 시나리오를 포함한 밸런스 설정 문서.

## 핵심 용어
- 장비 드랍률
- 장비 제작식
- 분해 재료
- 서버 인원 수 (AU 1.5만명, 동접 5000명)
- 컨텐츠 (필드 사냥, 일일 던전, 주간 던전, 필드 보스, 레이드 보스, 일일 미션, 서버 탐험)
- 장비 등급 (일반, 고급, 희귀, 영웅, 전설)
- 티어 (T1, T2, T3, T4)
- 장비 파편
- 은빛 큐브
- 크리스탈
- 차원석
- 흑색 메달
- 푸른빛 큐브 / 보라빛 큐브
- 설계도
- 코어
- 거래소
- IBT (Internal Beta Test)

## 숫자/상수/공식
- 서버당 AU 1.5만명 기준, 동접 5000명
- 컨텐츠 이용 서버군 5개
- 1인 1시간 장비 획득 3.6~6개 = 0.5% 드랍률
- 사냥 기준 3초~5초 주기 (쉬운 난이도 3초, 보통 난이도 5초)
- 획득 장비 중 20% 거래소 등록 가능
- 영웅 등급 무기 월 평균 5개, 방어구 파츠마다 월 5개, 장신구 월 1개
- 제작 재료 상위 환산: 기본 재료 10개 + 코어 1개 (또는 코어 3~10개)
- IBT 기간: 약 1주일, 일일 1~3시간 플레이 기준 (통상 1/5 감성)
- IBT 인원: 최대 200명 추정
