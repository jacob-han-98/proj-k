# 장비 아이템 위상 및 재료 획득처 (요약)

## 한 줄 설명
Project K의 장비 아이템을 등급과 티어로 구분하고, 각 등급/티어별 제작 재료 구성 및 획득처(필드, 던전, 보스, 상점 등)를 정의한 문서.

## 핵심 용어
- 등급 (Grade): 일반, 고급, 희귀, 영웅, 전설, 신화
- 티어 (Tier): 동일 등급 내 성능별 구분 (T1, T2, T3, T4)
- 파츠 (Parts): 무기, 방어구, 장신구 총 12종
- 제작 키 아이템: 크리스탈, 흑색 메달, 영웅 무기 설계도, 마력이 깃든 자수정, 황금 메달, 전설 무기 설계도, 마력이 깃든 토파즈
- 은빛 큐브: 기본 분해 재료
- 푸른빛 큐브: 희귀 분해 재료
- 보라빛 큐브: 영웅 분해 재료
- 차원석: 서버 탐험 필드 파밍 재료
- 장비 파편: 기본 제작 재료
- 필드 A/B/C/D/E/F: 난이도별 필드 지역
- 필드 보스: 만티코어, 스켈라, 로바르스, 소르브
- 주간 던전: 필드 난이도 제한 우회용 컨텐츠
- 인터서버 전장: 시즌 보상 기반 최상급 장비 획득처
- 레이드: 어스 드래곤, 블루 드래곤 (난이도별)
- 거래소: 다이아 기반 거래 시스템
- 배리에이션: 동일 티어 내 용도/드랍처별 스탯 차별화
- 대성공: 제작 성공 시 거래 가능 상태 획득
- 컬렉션: 고난이도 지역 전용 장비

## 숫자/상수/공식
- 총 파츠: 12개 (무기 1, 방어구 7, 장신구 4)
- 총 등급: 6가지 (일반, 고급, 희귀, 영웅, 전설, 신화)
- 일반 등급: T0~T3 (4단계)
- 고급 등급: T1~T3 (3단계)
- 희귀 등급: T1~T4 (4단계)
- 영웅 등급: T1~T3 (3단계)
- 전설 등급: T1~T2 (2단계)
- 신화 등급: 1종
- 거래소 가격 (다이아): 일반 10, 고급 15, 희귀 50, 영웅 1000, 전설 5000, 신화 10000
- 거래소 가격 (현금 원): 일반 125, 고급 187.5, 희귀 625, 영웅 12500, 전설 62500, 신화 125000
- 다이아 현금 치환가: 1 다이아 = 12.5원
- 영웅급 아이템 예상 현금가: 3~5만원
- 제작 세트 수량: 일반 1, 고급 3, 희귀 5, 영웅 4, 전설 2, 신화 1
