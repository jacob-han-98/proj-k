- 총 53개 콘텐츠 항목
- OBT 진행 항목: 16개 (성물, 성장요소4, 업적, 서버 스팟전, 공성전, 필드 보스 쟁탈전, 인터서버 보스, 길드 의뢰, 성주 던전, 길드 성장요소, 랭킹, 연대기, PVP 기록, 캐릭터 거래, 캐릭터 임대, BM 상점, 설정, 레드닷, 확률 고지, 도움말)
- 경제 밸런스 검증 대상: 51개 항목 (채팅, 월드맵 제외)
