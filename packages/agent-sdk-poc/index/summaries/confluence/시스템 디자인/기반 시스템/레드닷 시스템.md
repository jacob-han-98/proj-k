## 한 줄 설명
게임 내 유의미한 변화 발생 시 HUD에 레드닷을 표시하여 유저가 정보를 인지하고 활용할 수 있도록 하는 편의 시스템. 베이스 시스템 규정을 정립하고 향후 컨텐츠 확장에 대응하기 위한 기획.
