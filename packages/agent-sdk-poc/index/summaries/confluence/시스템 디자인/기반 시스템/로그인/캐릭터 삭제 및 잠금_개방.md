## 한 줄 설명
캐릭터 삭제 플로우, 슬롯 잠금/개방 규칙, 오프라인 모드 제한사항 및 관련 시스템 메시지를 정의하는 문서.

- 슬롯 개수
- 기본 제공 슬롯
- 잠금 처리 슬롯
- BM 상품
- 오프라인 모드
- 캐릭터 선택 창
- 대시보드
- 시스템 메시지
- 팝업 창
- ContentSetting
- 삭제 동의 문구
- 서버 접속
- 멀티 캐릭터 대시보드
- 획득 아이템
- 기간 만료
- 텍스트 키
- 툴팁

## 숫자/상수/공식
- 서버 당 총 4개의 슬롯 보유
- 기본 제공 슬롯 2개
- 잠금 처리 슬롯 2개
- 삭제 신청 후 N일 후에 자동 삭제 (ContentSetting에서 설정)
- 삭제 완료 횟수 1회마다 N일 증가 (ContentSetting에서 설정)
- 기간이 7일 미만 남을 경우 시스템 메시지 출력
