## 한 줄 설명
Project K의 캐릭터 생성 시스템 진행 플로우를 정의하며, 클래스 선택부터 닉네임 입력까지의 UI/연출 단계와 캐릭터 세부 정보창, 대표 스킬 미리보기 기능을 명시한 문서.

## 핵심 용어
- 클래스 통합 로비
- 캐릭터 고유 시퀀스
- 캐릭터 선택창
- 선택 캐릭터 세부 정보창
- 대표 스킬 미리보기 창
- 배경 연출
- 카메라 블렌드 처리
- idle 모션
- 닉네임 입력창
- Character 테이블
- SignatureSkill01_ID ~ 03_ID
- IconResource
- TextKeyTitle
- TextKeyDesc
- 워리어, 가디언, 아처, 아발리스터, 매지션, 샤먼

## 숫자/상수/공식
- 6인 캐릭터 (클래스 통합 로비에서 출력)
- 대표 스킬 3종
- SignatureSkill 04 데이터는 공란 유지
