# 멀티 캐릭터 시스템 (요약)

> 출처: 시스템 디자인 / 기반 시스템 / 멀티 캐릭터 시스템
> 원본: packages/confluence-downloader/output/시스템 디자인/기반 시스템/멀티 캐릭터 시스템/content.md

## 한 줄 설명
계정 내 여러 캐릭터를 동시에 육성할 수 있으며, 한 캐릭터가 오프라인 모드로 플레이 중이어도 다른 캐릭터를 플레이 가능한 시스템.

## 핵심 용어
- 멀티 캐릭터
- 오프라인 모드
- 캐릭터 슬롯
- 대시보드
- 자동 전투
- 서포트 모드
- BM (비즈니스 모델)
- 캐릭터 교체
- 오프라인 모드 남은 시간
- 오프라인 모드 상태
- 인벤토리 상태
- 물약 상태
- 사망 여부
- 필드 이름
- 획득 경험치
- 획득 골드
- ContentSetting
- 타 서버 플레이 제한
- PK 어뷰징 방지

## 숫자/상수/공식
- 기본 캐릭터 슬롯: 서버 별 2개
- BM 추가 슬롯: 최대 2개 추가 가능 (최대 4개 슬롯)
- 오프라인 모드 플레이 최대 개수: 4개 (계정 당)
- 물약 경고 기준: 50개 이하
- 인벤토리 경고 기준: 90% 이상
- BM 캐릭터 슬롯 구매 제한: 서버 별 2회

## 참조 시스템
- 오프라인 자동 사냥 시스템
- 캐릭터 삭제 및 잠금/개방
- ContentSetting.xlsx
