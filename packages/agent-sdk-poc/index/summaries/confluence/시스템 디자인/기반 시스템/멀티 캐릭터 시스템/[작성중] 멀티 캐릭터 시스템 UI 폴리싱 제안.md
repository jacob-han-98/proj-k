## 한 줄 설명

## 숫자/상수/공식
- 오프라인 모드 최대 캐릭터 수: ContentSetting.xlsx의 <OfflineModeMaxCount> 참조
- 캐릭터 리스트 최대 출력 개수: 4개
- 물약 경고 기준: 50개 이하
- 인벤토리 경고 기준: 90% 이상 또는 슬롯 맥스
- 인벤토리 슬롯 색상 변경 규칙: 90% 미만(흰색), 90% 도달(주황색), 100% 도달(빨간색)
- 자동 사냥 경과 시간 기준: 30초 이상 시 시간당 경험치 표시
- 캐릭터 항목 출력 우선순위: 접속 중 > 결과 확인 필요 > 오프라인 진행 중 아님 > 남은 시간 많음 > 레벨 높음
