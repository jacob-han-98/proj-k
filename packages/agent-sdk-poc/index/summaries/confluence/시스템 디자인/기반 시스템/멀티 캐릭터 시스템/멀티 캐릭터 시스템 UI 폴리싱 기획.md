## 한 줄 설명
멀티 캐릭터 시스템의 대시보드 및 정보 팝업 UI를 통해 여러 캐릭터의 상태를 한눈에 확인하고, 빠른 캐릭터 전환 및 오프라인 모드 관리를 지원하는 UI 폴리싱 기획서.

## 숫자/상수/공식
- 오프라인 모드 최대 개수: ContentSetting.xlsx의 <OfflineModeMaxCount> 참조
- 캐릭터 리스트 최대 출력 개수: 4개
- 체력 물약 경고 기준: 50개 이하
- 인벤토리 경고 기준: 90% 이상 또는 슬롯 맥스
- 인벤토리 슬롯 색상 변경 규칙: 90% 미만(흰색), 90% 도달(주황색), 100% 도달(빨간색)
- 경과 시간 표기 단위: 초, 분, 시간, 일
- 시간당 경험치 표기: 1만 미만(400/h 형식), 1만 이상(1만/h 형식), 1억 이상(1억/h 형식)
