## 한 줄 설명
온라인 및 오프라인 자동 사냥 모드에서 스케줄러에 의해 자동으로 퀘스트, 던전, 사냥터 등의 컨텐츠를 진행하는 시스템으로, 자동 전투 및 자동 정비 기능과 연동된다.

## 숫자/상수/공식
- 기본 자동 정비 이용 시간: 8시간
- BM 구매 시 추가 확장: 16시간
- 최대 자동 정비 이용 시간: 24시간
- 일일 던전 시간 설정 범위: 최소 1시간, 최대 24시간
- 사망 횟수 설정 범위: 최소 1회, 최대 10회
- 빠른 부활 활성화 시간: 5초 후 자동 부활 (기본 30초 대기 중)
- 위치 기억 미활동 시 중단 조건: 5분 동안 아무 행동 없음
- 서포트 모드 개방 조건 레벨: Lv.30 (주간 던전 개방 레벨)
- All 버튼 시간 계산: 기본 시간 + 충전 시간을 올림처리
