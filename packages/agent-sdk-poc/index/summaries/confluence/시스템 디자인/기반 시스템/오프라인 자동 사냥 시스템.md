## 한 줄 설명
유저가 게임을 종료해도 캐릭터가 접속한 채로 일정 시간 동안 자동 사냥을 진행하는 시스템으로, 수면/출근 등 관리할 수 없는 시간에 경손실을 방지하고 배터리를 절약하는 편의 기능.

## 숫자/상수/공식
- 기본 오프라인 모드 이용 시간: 8시간
- BM 구매 확장 시간: +16시간 (총 24시간 가능)
- 매일 초기화 시점: ContentSetting의 DailyResetTime 기준
- 예상치 계산 기준: 30초 동안 획득한 경험치/골드 * 120 / h
- 물약 소모 계산: [남은 물약 개수 / 30s 동안 사용한 물약 개수 * 30] 초
- 오프라인 모드 진입 카운트: 3초
- 경험치 표시 소수점: 넷째 자리까지
