## 한 줄 설명
Project K의 우편 시스템 UI/UX 디자인 스펙을 정의하는 문서로, 우편함 호출 UI, 우편 목록 화면, 상세 내용 팝업, 첨부물 지급 UI 등의 구성과 동작 방식을 명시한다.

## 숫자/상수/공식
- 최대 우편 개수: 100개
- 우편함 정리 알림 기준: 90% 이상 (90개)
- 첨부물 슬롯 제한: 5개
- 첨부물 지급 UI 배치: 5열
- 첨부물 정상 표시: 10개 (2열)
- 청약철회 기간: 대체적으로 14일
