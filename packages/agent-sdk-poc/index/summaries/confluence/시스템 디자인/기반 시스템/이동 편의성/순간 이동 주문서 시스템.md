## 한 줄 설명
플레이어가 소모성 아이템(순간 이동 주문서)을 사용하여 현재 월드 내의 무작위 이동 가능 좌표로 즉시 워프되는 시스템으로, 사냥 효율 최적화와 전투 변수 창출을 목적으로 한다.

## 핵심 용어
- 순간 이동 주문서
- 무작위 좌표
- NavMesh
- 이동 가능 영역
- WorldTerritory
- TeleportRandom
- BuffClass
- Effect 상태
- TeleportDisable
- AllActionDisable
- 재사용 대기시간
- 소모성 아이템
- 퀵슬롯
- 인벤토리

- 펫
- 소환수
- 무적 상태
- 플레이어 리스폰 공통 규칙
- ContentSetting
- SystemMsg

## 숫자/상수/공식
- 최대 중첩: 9,999개
- 재사용 대기시간: 1.0초
- 최대 재산출 횟수(TeleportRandomMaxRetry): 10회
- 무적 버프 지속시간: 1.0초 (폐기됨, 플레이어 리스폰 공통 규칙으로 변경)
