## 한 줄 설명
플레이어가 월드 내 특정 좌표를 DB에 저장하고 전용 UI를 통해 즉시 이동하는 편의 시스템으로, 저장은 무료이며 이동 시 축복받은 이동 주문서 1개를 소모한다.

## 핵심 용어
- 위치 기억 시스템
- 기억 슬롯
- 축복받은 이동 주문서
- 저장 허용 지역
- 위치 기억 목록 팝업
- 컨텍스트 메뉴
- 위치 공유
- 하이퍼링크
- 재사용 대기시간(Cooldown)
- 북마크 텔레포트
- WorldTerritory
- ContentSetting
- BookmarkSave
- 안전 좌표 보정
- NavMesh
- 플레이어 리스폰 공통 규칙
- 무적 상태
- 채팅 링크
- 금칙어 필터
- 말줄임표

## 숫자/상수/공식
- 기본 기억 슬롯: 10개
- 이동 비용: 축복받은 이동 주문서 1개
- 저장 비용: 무료(Free)
- 재사용 대기시간: 1.0초
- 아이템 최대 중첩: 9,999개
- 슬롯명 최대 글자수: 12자
- 한 화면 최대 표시 슬롯: 5~10개
- 정렬 규칙: 최신순(저장 시간 역순)
