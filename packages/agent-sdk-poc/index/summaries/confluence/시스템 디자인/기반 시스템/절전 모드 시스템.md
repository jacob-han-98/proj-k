## 한 줄 설명
수동 조작 중단 시 기기 보호와 전력 소모 최소화를 위해 진입하는 상태로, 성장/보상 정보 모니터링과 채팅 사용을 지원한다.

## 숫자/상수/공식
- 진입 자동 시간: 1분/3분/5분/사용 안 함 (기본값 3분)
- 프레임 제한: 15fps~20fps 이하
- 경험치 계산 기준: 30초 동안 획득 경험치 기반
- 경험치 표시 소수점: 넷째 자리까지
- 인벤토리 색상 변경 기준: 90% 미만(흰색), 90% 도달(주황색), 100% 도달(빨간색)
- 시간 당 경험치 표기: 1만 미만(400/h), 1만 이상(1만/h), 1억 이상(1억/h)
