## 한 줄 설명
레퍼런스 게임(뱀피르, 리니지W)의 환경설정 옵션을 조사하여 Project K에 적용할 수 있는 전투, 편의, 환경, 알림 관련 설정 항목들을 정의한 문서.

## 숫자/상수/공식
- 자동 전투 범위 내 대상 감지
- 전투 기록 저장 조건: 1분 이상 / 3분 이상 / 5분 이상 / 10분 이상
- HP 물약 자동 사용 비율: 0~100 (기본값 80)
- PVP 시 자동 사용 HP 비율: 0~100 (기본값 100)
- 절전 모드 설정: 3분 / 5분 / 10분 / 사용 안함
- 화면 재생 속도: 15fps / 30fps / 60fps / 제한 없음
- 카메라 회전 속도: 0-100
- UI 배율: 50% / 75% / 100% / 125% / 150%
- 마우스 커서 크기: 100% / 150% / 200%
- 플레이어 표시 인원 제한: 30 / 50 / 100 / 200 / 무제한
- 사운드 볼륨: 0~100
