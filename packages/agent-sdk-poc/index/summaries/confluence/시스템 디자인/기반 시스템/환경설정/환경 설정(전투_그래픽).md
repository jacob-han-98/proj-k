## 한 줄 설명
Project K의 환경 설정 시스템에서 전투 및 그래픽 옵션을 정의하는 문서로, 모바일 포팅을 위해 그래픽 설정을 우선 구현하며 자동 전투, 획득, 물약, 연출 등의 전투 관련 설정과 화면, 사운드 설정을 포함한다.

## 핵심 용어
- 자동 전투 모드 설정
- 사냥 모드(PvE)
- 적대 모드
- 살의 모드
- 매너 사냥
- 몬스터 자동 반격
- 공격 대상 우선 순위 설정
- 전리품 획득 등급
- HP 물약 자동 사용
- 그래픽 간편 설정
- 화면 재생 속도(fps)
- 수직 동기화(Vsync)
- 외형 간소화
- 애니메이션 최적화 설정
- 플레이어 표시 인원 제한
- 이펙트 최소화 설정
- 포스트 프로세싱
- 안티 앨리어싱(TSR 방식)
- 모델링 품질
- 그림자 품질
- 전역 조명
- 반사 품질
- 텍스처 품질
- 식생 품질(Foliage)
- 사운드 볼륨

## 숫자/상수/공식
- 자동 전투 범위: 최소 5m, 최대 40m, 기본값 20m
- 전투 기록 저장 조건: 1분 이상 / 3분 이상 / 5분 이상 / 10분 이상
- HP 물약 자동 사용 비율: 0~100 (기본값 80)
- PVP 시 자동 사용 HP 비율: 0~100 (기본값 100)
- 화면 재생 속도: 15fps(낮음) / 30fps(중간) / 60fps(높음) / 120fps(매우 높음) / 무제한
- 플레이어 표시 인원 제한: 30 / 50 / 100(기본값) / 200 / 무제한
- UI 배율: 50% / 75% / 100% / 125% / 150%
- 마우스 커서 크기: 100% / 150% / 200%
- 사운드 볼륨: 0~100 (기본값 100)
- 절전 모드 설정: 3분 / 5분 / 10분 / 사용 안함(기본값)
