## 숫자/상수/공식
- 길드 이름: 최대 8자(영문 16자)
- 길드 소개: 최대 16~20자
- 길드 제작 최소 재료: 생산 재료 10개 / 제작 완제 1개
- 길드 제작 효과 곡선: 지수함수 형태
- 협동 보스 최소 고투력: 3~4인 이상
- 협동 보스 저투력 필요 인원: 12~15인 이상
- 길드 레벨별 컨텐츠 개방: 거점 점령전(3) / 협동 토벌전(4) / 영지 방어전(6) / 길드 제작(8) / 길드 쟁탈전(10) / 길드 공성전(13)
- 길드 레이드 주간 횟수: 주 1회 또는 주 2회
- 길드 기부 횟수: 각 3회(골드/모리온/다이아)
- 길드 기부
