# 길드 금고 시스템 (요약)

## 한 줄 설명
길드 단위로 획득한 재화(다이아)와 아이템을 보관하고 길드원들에게 분배하는 시스템으로, 아이템 분배와 다이아 분배(개별/일괄)의 두 가지 플로우를 제공한다.

- 설정 분배금

## 숫자/상수/공식
- 보관 최대 개수: 100개
- 금고 공간 부족 판정: 5칸 미만
- 일일 다이아 분배 제한: 100,000 다이아(약 27만원 가치)
- 아이템 분배 기록 보관일: 30일
- 균등 분배 나머지 처리: 기여도 순서대로 1 다이아씩 추가
- 기여도 분배 공식: 개인별 주간 기여도 / 선택 인원 총 기여도 × 분배금액
