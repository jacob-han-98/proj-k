## 한 줄 설명
같은 뜻과 목적을 가진 사용자들이 모여 몬스터 사냥 및 전투를 함께하며 길드 혜택을 공유하고 배틀 커뮤니티를 활성화하는 시스템.

## 숫자/상수/공식
- 길드 메뉴 개방 조건: 레벨 15 이상
- 길드 이름: 2자~8자, 중복 불가
- 길드 소개: 최대 20자
- 추천 길드 표시: 최대 30개
- 신청/초대 목록: 최대 30개
- 신입 길드원 유지 기간: 24시간
- 길드 공지사항: 최대 4줄, 최대 100자
- 길드 연혁 로그: 최대 100개
- 적대/우호 길드: 최대 20개
- 길드 초대 최대 개수: 20개
- 가입 신청 최대 개수: 100개
- 길드 해산 대기: 24시간
- 길드 탈퇴 후 대기: 24시간
- 길드장 미접속 자동 위임: 14일 이상
- 추천 길드 최소 출석율: 30% 이상 (0.3 <= 전날 출석 인원/현재 가입한 길드원)
- 기본 정렬 우선순위: 가입 유형(즉시 가입=2, 승인 필요=1, 가입 불가=0) → 주간 점수 내림차순 → 전날 출석 인원 내림차순 → 길드 레벨 내림차순 → 길드원 내림차순 → 길드 랭킹 오름차순 → 길드 이름 가나다순
- 길드원 정렬 우선순위: 직책(길드장>관리자>길드원>신입길드원) → 누적 기여도 내림차순 → 주간 기여도 내림차순 → 전투력 내림차순 → 접속 상태 → 출석 → 레벨 내림차순 → 닉네임 순
- 자동 추방 미접속일: 최소 1일, 최대 30일
- 자동 추방 주간 기여도: 최소 1점, 최대 10만점
- 길드명 변경 쿨타임: 7일
