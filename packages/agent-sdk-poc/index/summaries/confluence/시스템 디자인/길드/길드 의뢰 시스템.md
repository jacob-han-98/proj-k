# 길드 의뢰 시스템 (요약)

> 출처: 시스템 디자인 / 길드 의뢰 시스템

## 한 줄 설명
길드원이 함께 재료를 투자하여 공방에 아이템을 제작 의뢰하고 완성된 아이템을 분배받는 협력 제작 시스템으로, 서버 전쟁 보급과 개인 성장을 목표로 한다.

- 길드 의뢰
- 길드 제작
- 재료 투입

- 시간 가속
- 가속권
- 가속권 마일리지
- 제작 슬롯
- 길드장
- 길드원
- 컬렉션 아이템
- 패시브
- 길드 스킬
- 서버 기여도
- 공성전
- 공성 구조물
- 성문 강화
- 방어탑 강화
- 의뢰 이력

## 숫자/상수/공식
- 길드레벨 8: 제작 슬롯 1개 개방
- 길드레벨 14: 제작 슬롯 추가 확장
- 길드레벨 20: 제작 슬롯 추가 확장
- 제작 횟수 제한: 아이템마다 설정 가능, 공성전 종료 시 초기화
- 마일리지 초기화: 매주 월요일 오전 5시
- 의뢰 이력 보관: 최대 200개, 최대 30일
- 서버 기여도 계산: 총 보상 × 아이템별 책정 비율 × 서버 기여도
- 예시 계산: 50000다이아 × (1000/10000) × (534/10000) = 267다이아
- 전투 식량 스택 단계: 100개 이상 1단계, 500개 이상 2단계, 1000개 이상 3단계
- 성문 강화 제작 횟수 제한: 10회
- 방어탑 강화 제작 횟수 제한: 5회
- 공성 강화 제작 횟수 제한: 1회
- 컬렉션 아이템 2 제작 횟수 제한: 10회
- 컬렉션 아이템 3 제작 횟수 제한: 10회
