## 한 줄 설명
클리어한 무한의 탑 층에 정령을 배치하여 오프라인 상태에서 시간 비례 보상을 누적하는 방치 사냥 시스템. 정령 육성 결과를 생산성으로 환원하고 일일 방문 루틴을 형성한다.

## 핵심 용어
- 점령(Occupation)
- 슬롯(Slot)
- 팀 전투력(TeamCP)
- 요구 전투력(RequiredCP)
- 불안정 상태(Unstable State)
- 누적 시간(AccTime)
- 최대 누적 시간(AccTimeCap)
- 보상 계산 주기(RewardTickUnit)
- 불안정 상태 체크 주기(UnstableTickUnit)
- 불안정 상태 페널티 주기(UnstableTickPenalty)
- 기본 이벤트 발생 확률(BaseEventProb)
- 보너스 이벤트 발생 확률(BonusEventProb)
- CPRatio
- 보상 효율(RewardEfficiency)
- 불안정 발생 확률(UnstableProb)

- 역할 코어(Role Core)

## 숫자/상수/공식
- 기본 슬롯 개수: 1
- 최대 슬롯 개수: 3
- 기본 보상 계산 주기(RewardTickUnit): 10분
- CPRatio = TeamCP / RequiredCP
- UnstableProb = min(1, (1 - CPRatioUnstableParam) × (1 - StableRole))
- RewardTick = floor((NowServerTime - LastOccupyTime) / RewardTickUnit) + (RewardRole 성공 시 1)
- UnstableRewardTick = min(0, floor((LastUnstableTime - LastOccupyTime) / UnstableTickUnit) - UnstableTickPenalty)
- EventProb = BaseEventProb × CPRatio × BonusEventProb × LuckyRole
- AbortRewardTick = min(0, floor((NowServerTime - LastOccupyTime) / RewardTickUnit) - AbortTickPenalty)
- 정령 획득 경험치 = ExpPerMin × floor(NowServerTime - LastOccupyTime) × (1 + GrowthRole)
