## 한 줄 설명
24시간 상시 운영되는 인터서버 전용 던전 '공허의 틈새'의 리뉴얼 시스템을 정의하는 기획서로, 이원화된 재화 시스템과 서버 단위 경쟁을 강화하는 전면 개편 방향을 제시한다.

## 숫자/상수/공식
- 하층: 3개 서버 그룹
- 상층: 6개 서버 (2개 그룹 통합)
- 연속 처치 쿨타임: 5분
- 유효 오차 범위: 공격자 전투력의 70% 이상
- 주간 획득 한도: 통합 게이지 공유, 매주 월요일 04시 초기화 (예시)
- 전투력 비례 계산: 공격자와 방어자의 CP 차이 대조
