## 한 줄 설명
도깨비와 함께 전투하는 클리어&랭킹 기반 1인 인스턴스 던전으로, 최대 100층의 시즌제 컨텐츠이며 층별 클리어 보상과 최종 랭킹 보상을 제공한다.

## 숫자/상수/공식
- 최대 층수: 100층
- 각 층 전투 제한 시간: 90초
- 각 층 입장 후 스테이지 시작 대기 시간: 10초
- 1~75층: 15층 단위 반복 구성, 5층 단위 보스전
- 76~100층: 변별력 강화 구간, 90층부터 난이도 급상승
- 주간 무료 입장 횟수: 30회 (1개월 기준)
- 4주 기준 도전/클리어 가능: 120회
- 전용 버프 보상 최대 선정 개수: 3개
- 유저 수동 선택 버프: 1개
- DB 저장 랭킹: 1~100위
- 시즌 보상 지급 시점: 시즌 종료 시간 이후 10분 뒤
- 클리어 타임 저장 대상: 100층에서만 저장
- 동일 층 여러 번 클리어 시: 가장 짧은 시간 저장
