## 한 줄 설명
초식/육식 유저의 플레이 목적을 분리하면서 공존 가능한 인터서버 던전 생태계를 구축하고, 보스 몬스터를 중심으로 서버 단위의 진영전(RVR)을 유도하는 고도화 기획안.

## 숫자/상수/공식
- 하층 입장 조건: 35레벨
- 상층 입장 조건: 50레벨
- 구역별 최대 수용 인원: 1000명
- 하층 서버 그룹: 3개 서버
- 상층 서버 그룹: 6개 서버
- 동일 유저 반복 처치 쿨타임: 5분 (예시)
