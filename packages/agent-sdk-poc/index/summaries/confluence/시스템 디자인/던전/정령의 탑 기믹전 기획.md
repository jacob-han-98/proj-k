# 정령의 탑 기믹전 기획 (요약)

## 한 줄 설명
정령의 탑 던전에서 구현할 기믹전 시스템(수호 오브젝트, 환경 기믹, 특수 패배 조건, 전투 시작 딜레이)의 기능 개발 요소를 정의한 기획 문서.

## 핵심 용어
- 정령의 탑
- 수호 오브젝트
- 오브젝트 타입 (Object)
- 더미 타입 (Dummy)
- 환경 기믹
- 함정
- 특수 패배 조건
- 타임 어택
- 웨이브 전투
- DeathAction
- MonsterTypeEnum
- BattleObjectTypeEnum
- AggroTargetingTypeEnum
- ProtectTower
- InfiniteTowerFloor
- InfiniteTowerStageStartWaitTime
- 피아 식별
- 어그로
- Ready 상태
- Battle 상태
- 타겟팅 (Auto, Manual, Untarget)

## 숫자/상수/공식
- InfiniteTowerStageStartWaitTime: 1 = 1초 (제안값: 3초)
- MonsterTypeEnum - ProtectTower: 7
- BattleObjectTypeEnum - ProtectTower: 14
- AggroTargetingTypeEnum - OnlyProtectTowerTarget: 9
- AggroTargetingTypeEnum - OnlyCompanionTarget: 8

## 참조 시스템
- 무한의 탑 시스템
- MonsterBattleClass
- BattleConditionType
- MonsterSkill 테이블
- MonsterClass 테이블
- ContentSetting 테이블
- Buff 테이블
- BattleObjectTypeRelation 테이블
- TriggerEnum

- 개요
