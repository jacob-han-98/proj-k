## 한 줄 설명
Project K의 경제 시스템 설계 방향으로, 리니지식 P2W 기반의 계층화된 경제구조와 무소과금 플레이 가능성, 그리고 인플레이션 억제를 위한 재화 순환 구조를 정의한다.

## 핵심 용어
- 리니지라이크
- 가치 보전
- 경제적 역할 (상위-중위-하위)
- 아이템 흐름
- 화폐 흐름
- 계층 상승
- 무소과금
- PvP / PvE
- 재화 하수구
- 인플레이션 억제
- 가치의 상대적 위치
- 동적 서버
- 능동적 콘텐츠 업데이트
- 고가치 아이템 수량제한
- 능력치 인플레 지양
- 전투/성장/경제 순환구조
- 베이스 재화
- 종적 성장 / 횡적 성장
- 전문기술 / 공방
- 순환 가치 / 보존 가치
- 고래/돌고래 유저
- 변신 / 정령(펫)
- 강화 / 교체

## 숫자/상수/공식
- 상위 6~7% 고래/돌고래 유저 (매출비중 80% 이상)
- 무소과금 플레이 제한: PvP 콘텐츠는 넘을 수 없는 허들, PvE 콘텐츠는 시간과 노력으로 접근 가능
- 경제 밸런스 원칙: 생산 < 소비 (불균형 유지)
