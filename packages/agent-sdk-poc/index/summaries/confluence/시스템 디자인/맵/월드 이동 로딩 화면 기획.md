# 월드 이동 로딩 화면 기획 (요약)

> 출처: 시스템 디자인/맵
> 원본: packages/confluence-downloader/output/시스템 디자인/맵/월드 이동 로딩 화면 기획/content.md

## 한 줄 설명
월드(+인스턴스) 진입 시 월드별 지정된 로딩 화면(원화, 텍스트)을 랜덤 출력하고 프로그래스 바로 로딩 진행률을 시각화하는 시스템.

## 핵심 용어
- WorldLoading 테이블
- WorldClassId
- ArtworkPath
- Title
- Description
- Contents
- ContentsTypeEnum
- 프로그래스 바
- 레터박스 처리
- 원화
- 공용 로딩화면
- 인스턴스
- UIString
- textkey
- 게이지 바
- 로딩 진행률

## 숫자/상수/공식
- WorldClassId = 0: 공용 로딩화면 용도
- ContentsTypeEnum: 0 = None, 1 = Explore, 2 = Attack
- 프로그래스 바: 100분율로 나눈 뒤 맵 로딩 상황에 따라 1단위로 증가
- 로딩 텍스트: "월드를 로딩 중입니다 {0}%" (1단위 숫자 증가)
