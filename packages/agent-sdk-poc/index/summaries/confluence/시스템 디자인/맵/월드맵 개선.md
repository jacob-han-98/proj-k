# 월드맵 개선 (요약)

> 출처: 시스템 디자인 / 맵 / 월드맵 개선
> 원본: packages/confluence-downloader/output/시스템 디자인/맵/월드맵 개선/content.md

## 한 줄 설명
기존 월드맵 시스템을 바탕으로 대륙맵 화면의 국가 아이콘, 버튼, 클릭 처리 기능을 정식 버전으로 구현하는 작업을 정의한다.

## 핵심 용어
- 월드맵
- 대륙맵
- 중간맵
- 국가 아이콘 (flag)
- WorldCountry 테이블
- WorldTerritory 테이블
- RegionType
- Town
- Product 컬럼
- flag 컬럼
- textkey
- 위젯
- 좌표 (X, Y)
- Offset 이동
- SystemMSG
- toast 메시지
- WorldCountryId
- WorldRegionTypeEnum

## 숫자/상수/공식
- id값(WorldTerritory 테이블)이 낮은 마을을 대상으로 선정 (마을 2개 이상일 경우)
- 중앙 영역을 기준으로 X 또는 Y축 값을 Offset 이동하여 리소스 내 최대 가용 범위로 출력

## 참조 시스템
- PK_월드맵 시스템.xlsx (월드 및 지역 표시 시트)
