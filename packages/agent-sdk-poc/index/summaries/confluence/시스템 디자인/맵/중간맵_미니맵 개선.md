## 숫자/상수/공식
- 줌 단계: 기존 4단계 → 3단계로 변경 예정
- 이동 동선 간격: 5m 단위 (임시 값)
- 거리 단위: 500미터 당 텔레포트 비용 계산
- 대륙이동 텔레포트 비용 = TeleportFeeContinental + TeleportBaseFee + (TeleportFeePerDistance × 거리[500미터 당])
- 대륙 간 이동 NPC: 대륙 당 1명
