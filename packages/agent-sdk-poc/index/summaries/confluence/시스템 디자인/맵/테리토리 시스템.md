월드의 일부 또는 전체에 특정 기능/효과를 적용하기 위해 영역을 지정하고 관리하는 시스템으로, 기획 용도에 따라 월드에 적용되는 기준을 제공한다.

## 숫자/상수/공식
- ID 범위: 1~200 (일반 필드), 201~999 (던전), 1000~9999 (필드 거점)
- 경계 지역 in/out 버퍼: 약 1.5초
- 리스폰 볼륨 ID 중복 시: 랜덤한 볼륨 위치에서 리스폰
- RespawnVolumeId 배열: 길이 기반으로 가장 가까운 볼륨으로 이동
