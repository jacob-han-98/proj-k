# 몬스터 소환 (요약)

> 출처: 시스템 디자인 / 몬스터 / 몬스터 소환
> 원본: packages/confluence-downloader/output/시스템 디자인/몬스터/몬스터 소환/content.md

## 한 줄 설명
몬스터가 스킬을 통해 목표 위치에 다른 몬스터를 스폰하는 기능을 정의한다. 모체 몬스터 사망 시 소환된 몬스터의 디스폰 여부를 제어할 수 있다.

- SummonMonster
- SkilCategoryEnum

- Area Location
- 히트
- 스폰
- 디스폰
- 모체 몬스터
- Option
- 단일 몬스터 ID
- 랜덤 스폰 범위
- 박스

## 숫자/상수/공식
- 0 = False(기본값, 모체 몬스터 사망 시 소환 몬스터 유지)
- 1 = True(모체 몬스터 사망 시 소환 몬스터 디스폰)
- 100(1M, 랜덤 스폰 범위 예시)
- 4162(소환 몬스터 ID 예시)
- 25021(스킬 ID 예시)
