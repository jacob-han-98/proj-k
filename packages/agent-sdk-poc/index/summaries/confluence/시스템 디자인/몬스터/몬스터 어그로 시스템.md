## 한 줄 설명
몬스터가 전투 대상을 찾고 지정하는 기능으로, 대미지 누적 기반 우선순위와 선공/비선공 구분, 다양한 타겟팅 타입을 통해 어그로 대상을 결정한다.

## 숫자/상수/공식
- 어그로 수치 = 누적 대미지 (1:1 비율)
- 선공 몬스터 초기 어그로 수치: 1
- 타겟 갱신 쿨타임 기본값: 5초 (MonsterDefaultTargetRefreshTime 미입력 시)
- 사거리 예시: 3m, 6m, 8m, 9m
