## 한 줄 설명
보스 스케쥴 시스템의 UI/UX 디자인을 정의하는 문서로, HUD 아이콘, 스케쥴 리스트, 월드맵 연동, 토스트 메시지 등 플레이어 인터페이스 요소를 상세히 기술한다.

- 네임드 몬스터

- 텔레포트
- 출현 중

- 보상 아이템
- 디스폰
- 카운트 다운
- 툴팁

## 숫자/상수/공식
- 알림 설정 최대 4개
- 스폰 5분 전 알림창 출력
- 남은 시간 1일 이상: [00일 00시간] 표기
- 남은 시간 1일 미만: [00시간 00분] 표기
- 남은 시간 5분 미만: [00분 00초] 표기
- 등장 시간 표기: 00:00 형식
