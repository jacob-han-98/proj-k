## 한 줄 설명
변신 시스템을 개선하여 외형과 성능을 분리하고, 플레이 스타일 보정 중심으로 재설계하며, 합성 실패 누적 시스템과 수집 포인트 기반 스탯 트리를 도입하는 기획.

## 핵심 용어
- 변신 (Metamorph)
- 외형변경 (Skin)
- 공성전 아바타 (Siege)
- 메타모르프클래스 (MetamorphClass)
- 메타모르프포인트스탯 (MetamorphPointStat)
- 합성 실패 게이지 (Compose Fail Gauge)
- 수집 포인트 (Collection Points)
- 포인트 스탯 리스트 (Point Stat List)
- 등급 (Grade: Rare, Unique, Legendary)
- 클래스 정체성
- 플레이 스타일 보정
- 아바타 리스트
- 착용 상태
- 무기 클래스 제한
- 스킬 메타
- 성능 강요
- 누적되는 성장
- 천장 시스템
- 스킬북
- 패시브 스킬
- 평타 (Normal Attack)

## 숫자/상수/공식
- MaxComposeFailLegendary: 32 (전설 등급 실패 게이지 최대값)
- MaxComposeFailUnique: 128 (영웅 등급 실패 게이지 최대값)
- MaxComposeFailRare: 512 (희귀 등급 실패 게이지 최대값)
- 합성 실패 시 해당 등급 실패 게이지 +1
- 실패 게이지 = (현재값) / (최대값)
- 컬렉션 시스템 성장 비중: 약 1%
- 동일 등급 변신 4개 합성으로 상위 등급 획득 시도
