## 한 줄 설명
클래스 정체성을 유지하면서 변신을 플레이 스타일 선택 축으로 재설계하고, 성능과 외형을 분리하여 수집의 자유도를 높이는 개선 기획.

## 숫자/상수/공식
- 동일 등급 변신 4개 합성으로 상위 등급 획득
- 컬렉션 시스템 전체 성장 비중 약 1% 차지
- 안전 강화: 경험치 100% 획득
- 영웅 이상부터 합성 실패 시 포인트 적립
