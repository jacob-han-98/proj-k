## 숫자/상수/공식
- 클래스 당 무기 제한: 1종류
- 일반~영웅 등급 클래스당 무기: 4종
- 전설, 신화 등급 클래스당 무기: 3종
- 헤드 파츠: 클래스당 6종
- 전설급 고유 애니메이션: idle 2종, 이동 모션 1종, 기본 공격 2종, 궁극기 1종
- 전설급 고유 외형: 2종
- 신화급 고유 외형: 1종
