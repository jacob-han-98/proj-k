## 한 줄 설명
변신 메뉴 진입 시 캐릭터 외형을 고품질로 출력하고 유저가 회전/줌 조작으로 자유롭게 감상할 수 있도록 하는 변신창의 UI/카메라 시스템 및 조작 방식을 정의한 문서.

- 보간

## 숫자/상수/공식
- Default_Distance: 변신 창 진입 시 대상으로부터 카메라 거리
- ZoomIn_Distance: 최대 줌 인 거리
- ZoomOut_Distance: 최대 줌 아웃 거리
- Default_Offset: Vector (X, Y) - 기본 카메라 위치
- ZoomIn_Offset: Vector (X, Y) - 최대 줌인 시 카메라 위치
- ZoomOut_Offset: Vector (X, Y) - 최대 줌 아웃 시 카메라 위치
- FOV: float - 카메라 FOV 제어
- Zoom_Sensitivity: float - 줌인/아웃 속도
- Rotate_Sensitivity: float - 캐릭터 회전 속도
- 360도 지속 회전 가능
