## 한 줄 설명
캐릭터의 특정 액션(공격 피해량, 받은 피해량, 버프/디버프)에 스코어를 부여하여 랭킹화하고, 다수 참여 중형 이상 컨텐츠에서 순위별 차등 보상을 제공하는 시스템.

- 받은 피해량
- 적중 버프/디버프 횟수
- 랭킹
- 보상 지급
- 보스 레이드

## 숫자/상수/공식
- 기여도 공식 = ((공격 피해량 총합 * 전용 계수) * 적용 비율) + ((받은 피해량 총합 * 전용 계수) * 적용 비율) + ((적중 버프/디버프 총 횟수 * 전용 계수) * 적용 비율)
- 소숫점 이하 버림 처리
- 각 적용 비율의 총합 = 100%
- 예시: 가디언 26,385점, 아처 29,914점, 샤먼 27,132점 (40% 공격 피해량, 30% 받은 피해량, 30% 버프/디버프)
- 보상 UI 최대 4.5줄 표시 (한 줄 5개 아이콘)
