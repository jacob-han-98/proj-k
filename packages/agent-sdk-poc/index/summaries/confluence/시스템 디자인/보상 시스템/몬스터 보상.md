# 몬스터 보상 (요약)

> 원본: packages/confluence-downloader/output/시스템 디자인/보상 시스템/몬스터 보상/content.md

## 한 줄 설명
몬스터 사냥 시 획득하는 보상 구조와 지급 방식을 정의하며, Beta2 개선으로 기여도 기반 경험치/골드 분산 및 아이템 소유권 규칙을 명시한다.

## 핵심 용어
- 몬스터 보상
- Monster (개별 몬스터 특산물)
- Spawner (지역 공통 보상)
- Territory (지역 전역 공통 보상)
- 기여도
- 대미지 기여도
- 마지막 공격자
- 파티 보상
- MonsterClass
- MonsterSpawner
- SpawnExtraExp
- GoldReward
- MonsterReward
- MonsterRandomReward
- 아이템 소유권
- 경험치 분산
- 골드 분산
- 거리 기준 (20m)

## 숫자/상수/공식
- 파티 보상 지급 기준 거리: 20m
- MonsterRandomReward 확률 합계: 1,000,000 = 100%
- 경험치 획득량 = 경험치 × {공격자의 기여도(%)}
- 골드 획득량 = 골드 × {공격자의 기여도(%)}
- 소수점 올림 처리 (예: 3 exp를 4명이 때리면 각 1 exp)
