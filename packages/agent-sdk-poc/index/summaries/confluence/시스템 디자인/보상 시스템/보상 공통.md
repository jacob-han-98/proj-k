## 한 줄 설명
Project K에서 플레이어 캐릭터에게 다양한 경로를 통해 제공되는 보상의 획득 구조, 지급 방식, UI, 테이블 정의를 규정하는 보상 시스템 전반의 공통 규칙.

## 숫자/상수/공식
- 한 줄에 5개씩, 최대 2.5줄 (획득 아이템 출력 영역)
- 상단 5개, 하단 6개 = 총 11개까지 출력 (변신/펫 획득 UI)
- 11개 단위로 다음 페이지 넘김
- 기본값: "1개" (아이템 수량 선택 UI)
- +, -, Min, +10, +100, Max 버튼 (수량 조절)
