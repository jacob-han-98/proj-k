# 업적 시스템 (요약)

> 출처: 시스템 디자인 / 보상 시스템 / 업적 시스템
> 원본: packages/confluence-downloader/output/시스템 디자인/보상 시스템/업적 시스템/content.md

## 한 줄 설명
부캐릭터 육성을 권장하고 초반 국민 스펙 달성 및 골드 수급을 지원하는 카테고리별 업적 달성 시스템으로, 레벨/성장/아이템/재화/모험/전투 6개 카테고리의 업적 완료 시 변신·정령·장비·골드 등의 보상을 제공한다.

## 핵심 용어
- 업적(Achievement)
- 카테고리(Category)
- 레벨(Level)
- 성장(Growth)
- 아이템(Item)
- 재화(Currency)
- 모험(Adventure)
- 전투(Combat)
- 변신(Metamorph)
- 정령(Pet)
- 강화(Enchant)
- 합성(Compose)
- 카테고리 포인트(Category Point)
- 업적 그룹핑(Achievement Grouping)
- 진행 상태(Progress Status)
- 보상 받기(Receive Reward)
- 모두 받기(Receive All)
- 후속 업적(Next Achievement)
- 업적 진행 팝업(Achievement Progress Popup)
- 보상 획득 연출(Reward Acquisition Effect)

## 숫자/상수/공식
- 카테고리 종합 업적 보상: 업적 보상 수령 후 점수 보상 지급 (2회 클릭 필요)
- 보상 아이템 최대 표시 개수: 4개 (리스트), 10개 초과 시 2줄 표시 (연출)
- 카테고리 수: 6개 (Level, Growth, Item, Currency, Adventure, Combat)
- 카테고리 이미지: 6종 필요

## 참조 시스템
- EquipTypeEnum (장비 타입)
- GradeEnum (등급)
- CurrencyEnum (재화 타입)
- QuestCategoryEnum (퀘스트 카테고리)
- MonsterTypeEnum (몬스터 타입)
- WorldClass ID (월드/던전)
- MonsterClass ID (몬스터)
- Achievement 테이블 예시.xlsx
