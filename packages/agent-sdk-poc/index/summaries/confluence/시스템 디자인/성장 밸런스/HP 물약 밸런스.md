## 한 줄 설명
1초 쿨타임 HP 물약의 회복량을 레벨별로 정의하고, 동레벨/레벨차 사냥에서의 생존 리듬과 물약 소모 경제를 설계한 밸런스 문서.

## 핵심 용어
- DPS(L): PC와 몬스터의 레벨이 L일 때 PC가 받는 초당 피해량
- P(T): 물약 티어 T의 1회 회복량
- MaxHPS: 초당 회복량 (쿨타임 1초 기준)
- UseRate(L): DPS(L) / MaxHPS (필요 사용 빈도)
- 자동사냥 리듬
- 레벨 차이 (+5, -5)
- 구간형 티어
- 티어 경계선
- 물약 소모량
- 동레벨 사냥
- 레벨차 사냥
- 온보딩
- 성장/운영 구간
- 방어/회피/피해감소
- 제작/드랍/상점 테이블

## 숫자/상수/공식
- UseRate 목표 (동레벨): 초반 0.1~0.2, 중반 0.2~0.3, 후반 0.3~0.4
- 물약 쿨타임: 1초
- 레벨 구간별 회복량: 1~20(500), 21~30(750), 31~40(2500), 41~50(4250), 51~60(8000), 61~70(11250), 71~80(15000), 81~90(21250), 91~100(30000)
- HPS/DPS 비율: 1~30레벨(1.5), 31~50레벨(1.4), 51~70레벨(1.3), 71~80레벨(1.2), 81~100레벨(1.1)
- 테스트 기준: 물약 2000개 소모 시간
