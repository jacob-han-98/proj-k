## 한 줄 설명
릴리즈 버전까지 사용될 표준화된 경험치 성장 곡선을 정의하고, 레버 기반 밸런싱을 통해 예측 가능한 성장률을 산정하는 문서.

## 핵심 용어
- 경험치 테이블
- 성장 곡선
- 레버 기반 밸런싱
- Player Config
- ExpPerHour Config
- Dungeon Config
- Quest Config
- Daily Config
- Level-Exp
- 누적 플레이 타임 시뮬레이션
- Level Design-Level
- Tier
- Exponential growth
- 기본 경험치
- 추가 경험치
- 과금 분류
- 입장/퇴장 레벨

## 숫자/상수/공식
- 플레이어 타입: 캐주얼, 상위 10%, 상위 1% (3가지)
- 과금 분류: 무과금, 소과금, 중과금, 대과금 (4가지)
- 시뮬레이션 기간: 6개월
- Exponential growth: base exp, growth rate
