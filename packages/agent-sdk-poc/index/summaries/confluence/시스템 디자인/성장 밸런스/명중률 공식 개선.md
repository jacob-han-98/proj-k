# 명중률 공식 개선 (요약)

> 출처: 시스템 디자인 / 성장 밸런스 / 명중률 공식 개선
> 원본: packages/confluence-downloader/output/시스템 디자인/성장 밸런스/명중률 공식 개선/content.md

## 한 줄 설명
기존 명중률 계산에 레벨 차이를 반영하고 시그모이드 함수 기반 곡선형 확률 모델을 도입하여 전투 밸런스를 정교하게 조정하는 개선안을 정의한다.

## 핵심 용어
- 명중률
- 회피
- 레벨 차이
- 시그모이드 함수
- HitBasePercent
- HitSigmoidSlope
- LevelWeight
- StatWeight
- HitMinPercent
- HitMaxPercent
- MeleeAccuracy
- RangeAccuracy
- MagicAccuracy
- AllAccuracy
- MeleeEvade
- RangeEvade
- MagicEvade
- AllEvade
- AttackerStat
- VictimStat
- ContentSetting
- 곡선형 확률 모델
- 가중치

## 숫자/상수/공식
- HitBasePercent: 공격자와 피격자 능력치 동일 시 기대 명중 확률
- HitSigmoidSlope: 확률 곡선 기울기 조절 파라미터
- LevelWeight: 레벨 차이 영향 비중
- StatWeight: 스탯 차이 영향 비중
- HitMinPercent: 명중 확률 최소 한계값
- HitMaxPercent: 명중 확률 최대 한계값
