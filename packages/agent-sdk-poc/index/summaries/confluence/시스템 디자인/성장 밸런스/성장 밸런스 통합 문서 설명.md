## 한 줄 설명
Project K의 성장 및 보상 밸런스를 정량적으로 설계·검증하기 위한 엑셀 기반 통합 시스템으로, 몬스터·아이템 드랍의 자동화와 시뮬레이션을 제공한다.

## 숫자/상수/공식
- 몬스터 사냥 기준: 10초/마리 = 6마리/분 = 360마리/시간
- 성장 부스터 배율: 0.5배, 1배, 1.25배, 1.5배, 1.75배, 2배
- 1레벨 기준 타수 (PC → Mon: 3.5, Mon → PC: 35, PC → PC: 11.7)
- 100레벨 기준 타수 (PC → Mon: 6, Mon → PC: 62.3, PC → PC: 20)
- 레벨 차 타수 (100레벨 PC → 85레벨 Mon: 3.2, 85레벨 Mon → 100레벨 PC: 252.6, 100레벨 PC → 90레벨 PC: 12.8)
- 최종 대미지 목표값: 약 400
- Drop Start Level Penalty 예시: 10% 확률, Start 10, Prime 20, Penalty 0.1 → 레벨 10에서 1%, 레벨 15에서 5.5%, 레벨 20에서 10%
