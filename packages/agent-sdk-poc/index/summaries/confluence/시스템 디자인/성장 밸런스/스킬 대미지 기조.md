# 스킬 대미지 기조 (요약)

> 출처: 시스템 디자인 / 성장 밸런스 / 스킬 대미지 기조
> 원본: packages/confluence-downloader/output/시스템 디자인/성장 밸런스/스킬 대미지 기조/content.md

## 한 줄 설명
Project K의 전투 시스템에서 평타와 스킬의 시간 점유 비율(60:40)을 기반으로 스킬 쿨타임, 마나 소모, 대미지 배수를 설정하는 기획 기조 및 계산 방법론을 정의한다.

## 핵심 용어
- 평타 (Normal Attack)
- 스킬 (Skill)
- 시간 점유 비율 (Time Occupation Ratio)
- 쿨타임 (Cooldown)
- 시전 시간 (Cast Time)
- 마나 (Mana)
- 스킬 등급 (Skill Tier): 일반, 고급, 희귀, 영웅
- DPS (Damage Per Second)
- 마나 회복량 (Mana Recovery)
- 마나 소모량 (Mana Consumption)
- 대미지 배수 (Damage Multiplier)
- 임팩트 (Impact)
- 누킹 (Nuking)
- 유틸성 (Utility)
- 체감 효율 (Perceived Efficiency)
- 선형 스케일링 (Linear Scaling)

## 숫자/상수/공식
- 평타:스킬 시간 비중 = 6:4 (60:40)
- 기준 전투 시간 = 100초
- 평타 주기 = 1.5초
- 평타 시전 시간 = 0.5초
- 평타 시간 배분 = 33초
- 스킬 시간 배분 = 22초
- 일반 등급 스킬 쿨타임 = 30초 (100초 전투 중 6.6회 등장)
- 고급 등급 스킬 쿨타임 = 45초 (100초 전투 중 6.6회 등장)
- 희귀 등급 스킬 쿨타임 = 60초 (100초 전투 중 6.6회 등장)
- 영웅 등급 스킬 쿨타임 = 180초 (100초 전투 중 0회 등장)
- 스킬 시전 시간 = 1초 (기본 가정)
- 시전 시간 배수 증가 시 쿨타임 = 선형 스케일링 적용
- 스킬 대미지 배수 범위 = 1.5~4.5배
- 평타 대비 순간 대미지 배수 = 3~4배 (체감 임팩트 기준)
- 평타 대미지 배수 수식 = Base + 쿨타임 계수 × (스킬 쿨타임 / 평타 주기) + 마나 소모량 계수 × 마나 소모량
