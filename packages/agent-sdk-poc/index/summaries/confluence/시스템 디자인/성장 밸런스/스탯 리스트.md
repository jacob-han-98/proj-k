## 한 줄 설명
Project K의 모든 캐릭터 스탯을 한글명, 영문명, 연산 타입, 설명과 함께 정의한 전체 스탯 목록.

## 핵심 용어
- Attack (공격력)
- AllAttackRange (공격 사거리)
- AttackSpeed (공격 속도)
- SkillSpeed (스킬 시전 속도)
- AllAccuracy (명중)
- CriticalChance (치명타 확률)
- CriticalDamage (치명타 대미지)
- AllDamage (모든 대미지)
- MeleeDamage (근접 대미지)
- RangeDamage (원거리 대미지)
- MagicDamage (마법 대미지)
- PvpAllDamage (PvP 모든 대미지)
- Defence (방어력)
- AllEvade (회피)
- AllGuard (막기)
- CcAllResist (상태이상 저항)
- MaxHp (최대 HP)
- MaxMp (최대 MP)
- MoveSpeed (이동 속도)
- RegenHp (HP 자연 회복량)
