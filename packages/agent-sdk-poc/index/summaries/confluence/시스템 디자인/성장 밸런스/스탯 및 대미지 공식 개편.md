## 한 줄 설명
Project K의 대미지 공식 및 스탯 시스템을 개편하여 Attack vs Defense 비교 기반의 새로운 대미지 계산 방식과 이에 따른 스탯 추가/제거/변경을 정의한 문서.

- 유효 방어력
- 기본 대미지
- 스킬 대미지

- 일반 증가 배율

- 치명타 대미지

## 숫자/상수/공식
- 기본 대미지 = 유효 공격력 * 유효 공격력 / (유효 공격력 + 유효 방어력 + ContentSetting:DefenceModifier)
- 스킬 대미지 = 기본 대미지 * SkillMultiplier
- 대미지 증가 = 스킬 대미지 * (절대 증가 배율 + 일반 증가 배율 + 조건 증가 배율)
- 치명타 적용 대미지 = 대미지 증가 * 치명타 대미지
- 대미지 감소 = Max(0, 치명타 적용 대미지 * (1 - 절대 감소 배율) * (1 - 일반 감소 배율) - 추가 감소)
- 막기 감소 = Max(0, 대미지 감소 * (1 - 막기 대미지 감소)) = 최종 대미지
- ContentSetting:CriticalBaseDamage = 0.5
- ContentSetting:MaxDamageCheckRangePercent = 10 (기본값)
- ContentSetting:CCTimeReductionMinPercentage = 1 (CC 감소 최소 시간)
- ContentSetting:CCTimeReductionMaxPercentage = 0.2 (CC 감소 최대 시간)
- 유효 공격력 = 공격력 - 공격력 상쇄 + 무기 랜덤 공격력
- 유효 방어력 = Max(0, 방어력 - 방어력 상쇄)
- 무기 랜덤 공격력 = (MinWeaponAttack ~ MaxWeaponAttack 사이의 랜덤 값) * ContentSetting:WeaponModifierPercent
