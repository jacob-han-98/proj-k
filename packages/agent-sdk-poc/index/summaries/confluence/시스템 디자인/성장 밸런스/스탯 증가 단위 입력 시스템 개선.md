# 스탯 증가 단위 입력 시스템 개선 (요약)

> 출처: 시스템 디자인 / 성장 밸런스 / 스탯 증가 단위 입력 시스템 개선
> 원본: packages/confluence-downloader/output/시스템 디자인/성장 밸런스/스탯 증가 단위 입력 시스템 개선/content.md

## 한 줄 설명
기존 정수 기반 ValuePerPoint 방식을 개선하여 포인트 대비 스탯 증가 비율을 유연하게 설정할 수 있도록 NumberOfPoints 컬럼을 추가하는 시스템 개선안.

## 핵심 용어
- ValuePerPoint
- NumberOfPoints
- StatClass.xlsx
- 포인트
- 스탯
- 스탯 증가 단위
- 성장 비율
- 밸런스 조정
- int32

## 숫자/상수/공식
- 스탯 = int( 찍은 포인트 / NumberOfPoints ) * ValuePerPoint
- 예시: 5 포인트 당 2 스탯 (int(포인트/5) * 2)
- 포인트 1~10에 대한 스탯 증가량 테이블 제시
