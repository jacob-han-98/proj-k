# 아이템 등급 차별화 기조 (요약)

> 출처: 시스템 디자인 / 성장 밸런스
> 원본: packages/confluence-downloader/output/시스템 디자인/성장 밸런스/아이템 등급 차별화 기조/content.md

## 한 줄 설명
레벨 구간별 기준 등급을 설정하고 표준 밸런스 대비 가산/감산으로 아이템 등급을 차별화하여 콘텐츠 난이도를 안정화하고 밸런스 유지보수를 용이하게 하는 기조.

## 핵심 용어
- 표준 밸런스
- 레벨 구간
- 기준 등급
- 단계
- Common
- Uncommon
- Rare
- Unique
- Legendary
- Myth
- 고정 보너스 스탯
- 파워 스파이크
- 장비 자동 생성
- 정수형 스탯
- 비율형 스탯
- 헤비 유저
- 라이트 유저

## 숫자/상수/공식
- 레벨 1-10: Common 기준 등급
- 레벨 11-30: Uncommon 기준 등급
- 레벨 31-50: Rare 기준 등급
- 레벨 51-100: Unique 기준 등급
- 정수형 스탯 단계: +1
- 비율형 스탯 단계: 0.1%
- Common 기준 등급 시 Uncommon과의 차이: +1
- Uncommon 기준 등급 시 Common과의 차이: -1, Rare과의 차이: +1
- Rare 기준 등급 시 Common과의 차이: -2, Uncommon과의 차이: -1, Unique와의 차이: +1
- Unique 기준 등급 시 Common과의 차이: -3, Uncommon과의 차이: -2, Rare과의 차이: -1, Legendary와의 차이: +1, Myth와의 차이: +2

## 참조 시스템
- PK_성장 밸런스.xlsm (장비 자동 생성 시트)

## 주요 섹션
- 기획 의도 및 목적
- 기조
- 기본 개념
- 표준 밸런스
- 레벨 구간과 기준 등급
- 단계
- 등급 간 기본 차이
- 상위 등급의 고정/랜덤 보너스
