# 플레이어가 무기를 착용하지 않았을 때 처리 개선 (요약)

> 출처: 시스템 디자인 / 성장 밸런스
> 원본: packages/confluence-downloader/output/시스템 디자인/성장 밸런스/플레이어가 무기를 착용하지 않았을 때 처리 개선/content.md

## 한 줄 설명
무기 미착용 시 최종 대미지가 0이 되는 문제를 해결하기 위해 경고 메시지 UI를 추가하여 플레이어 혼란을 방지하는 개선안을 정의한다.

## 핵심 용어
- 무기 착용
- 대미지 계산
- 공격자 스탯
- 방어자 스탯
- 최종 대미지
- 무기 슬롯
- 경고 메시지
- 캐릭터 머리 위 메시지
- 시각적 피드백
- 맨손 전투 모션
- 비무장 idle 포즈
- 특수 효과
- 기본 공격력
- Min Damage
- Max Damage
- 게임 컨셉

## 숫자/상수/공식
- 무기 비중: 약 40% (대미지 계산 시 공격자 전체 스탯의 합 중)

## 참조 시스템
- PK-4567 (Jira 태스크)

## 주요 섹션
- 기획 의도 및 목적
- 시스템 기획
- 메시지 및 UI 알림 추가
- 전투 시작 시 경고 메시지
- 기타 고려된 방안들
- 방안1: 무기 없이 공격 불가 처리
- 방안2: 무기를 착용하지 않은 경우 시각적 피드백 제공
- 방안3: 기본 공격력 추가
