# [개인 성장] 승천의 인장 (요약)

> 출처: 시스템 디자인 / 성장 컨텐츠
> 원본: packages/confluence-downloader/output/시스템 디자인/성장 컨텐츠/[개인 성장] 승천의 인장/content.md

## 한 줄 설명
캐릭터 레벨 해금 조건에 따라 1~50단계를 순차적으로 강화하여 능력치와 특별 버프를 획득하는 직선형 성장 시스템으로, 실패 페널티 없이 기적의 재시도와 버프 재설정을 통해 지속적인 재화 소모를 유도한다.

## 핵심 용어
- 승천의 인장
- 인장 탭 (투사의 인장 등)
- 인장의 파편
- 찬란한 인장의 정수
- 기적의 재시도
- 특별 구간 (10, 20, 30, 40, 50단계)
- 특별 버프
- 특별 버프 풀 (Pool)
- 버프 레벨 (중복 합산)
- 버프 재설정
- 인장 노드
- 석판 (노드 배경)
- 특별 인장 효과 슬롯
- 성공 확률
- 2단계 점프 (대박 판정)
- SealInfo 테이블
- SealLevelData 테이블
- SealSpecialPool 테이블

## 숫자/상수/공식
- 최대 단계: 50단계
- 특별 구간: 10, 20, 30, 40, 50단계
- 기적의 재시도 확률: 약 5% (예시)
- 2단계 점프 확률: 약 0.1% (예시)
- 특별 인장 효과 슬롯: 5개
- 성공 확률 기준: 10000분율 (10000 = 100%)
- 난수 범위: 1~10000
- 특별 구간 스킵 방지: 2단계 점프 범위 내 특별 구간 포함 시 1단계만 상승

## 참조 시스템
- Effect 테이블 (EffectClassId 참조)
- 기존 버프 테이블 (buffId 참조)
- 일일 던전 (인장의 파편 획득처)
- 주간 던전 (찬란한 인장의 정수 획득처)

## 주요 섹션
- 개요 (명칭, 컨셉, 기획 목적, 핵심 특징, 기대 효과)
- 핵심 규칙 및 시스템 구조 (기본 설정, 아이템 구성, 강화 규칙, 특별 구간 규칙)
- UI 및 기능 설명 (진입 버튼, 메인 UI, 부분별 상세 설명)
- 프로세스 로직 (기본 강화, 특별 인장 효과 재설정, UI 상태 갱신)
- 흐름도 (인장 강화 버튼 조작)
- 연출 목록 (기본 강화 연출)
- 데이터 테이블 명 (SealInfo, SealLevelData, SealSpecialPool)
- 메시지 명세 (시스템 메시지, UI 메시지)
