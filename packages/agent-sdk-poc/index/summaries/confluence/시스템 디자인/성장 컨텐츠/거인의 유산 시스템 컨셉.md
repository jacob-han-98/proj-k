## 한 줄 설명
Project K의 신규 횡적 성장 컨텐츠로, 거인의 신체 부위를 테마로 한 랜덤 옵션 부여 시스템. 높은 성장 체감, 확률 기반 재미, 게임 내 재화 소모를 목표로 설계됨.

## 숫자/상수/공식
- 기본 해금 슬롯: 1~2개
- 거인 부위 총 개수: 7개 (왼쪽 눈, 왼쪽 귀, 빈 머리, 코, 입, 오른쪽 눈, 오른쪽 귀)
- BM 추가 슬롯 해금: 최대 2~3개 (폐기된 안)
