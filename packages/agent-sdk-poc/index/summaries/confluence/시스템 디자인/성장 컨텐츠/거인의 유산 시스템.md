# 거인의 유산 시스템 (요약)

> 출처: 시스템 디자인 / 성장 컨텐츠 / 거인의 유산 시스템
> 원본: packages/confluence-downloader/output/시스템 디자인/성장 컨텐츠/거인의 유산 시스템/content.md

## 한 줄 설명
메인 퀘스트 진행에 따라 거인의 부위를 해금하고 재화를 투입하여 확률적으로 능력치를 획득하는 횡적 성장 컨텐츠. 슬롯 잠금 기능을 통해 BM 다이아를 소모하여 원하는 능력치만 재시도할 수 있다.

## 핵심 용어
- 거인의 유산 시스템
- 거인의 머리
- 거인의 부위 (왼쪽 눈, 오른쪽 눈, 왼쪽 귀, 오른쪽 귀, 코, 입, 머리)
- 리프 정착
- 능력치 슬롯
- 1단계/2단계/3단계 해금
- 거인의 피 (확률업 아이템)
- 슬롯 잠금/고정
- 등급 (일반, 고급, 희귀, 영웅, 전설, 신화)
- 능력치 카테고리 (공격력 스탯, 방어력 스탯, 유틸 스탯)
- 골드 (필수 재화)
- 코인 (필수 재화, 구 명칭: 코어)
- BM 다이아 (슬롯 잠금용 재화)
- 메인 퀘스트
- 해금 조건
- 확률 상승

## 숫자/상수/공식
- 총 항목 수: 7개
- 항목별 능력치 슬롯: 3개 (1단계, 2단계, 3단계)
- 최대 부여 가능 능력치: 21개 (7 × 3)
- 능력치 등급: 6단계 (일반, 고급, 희귀, 영웅, 전설, 신화)
- 비주얼 이펙트 지속시간: 1.5~2초 이내
- 해금 단계: 3단계 (1차, 2차, 3차)

## 참조 시스템
- 횡적 성장 요소 시스템 컨셉
- 서버탐험 (코인 획득 컨텐츠)
- 메인 퀘스트 시스템

## 주요 섹션
- 개요 (목적, 특징/요약)
- 시스템 구현 (구조, 항목, 해금, 능력치 획득)
- 비주얼 컨셉/UI (컨셉, 클라이언트 처리 플로우 및 UI)
- 테이블 구성 (거인의 유산 항목 테이블, 획득 능력치 테이블, 등급별 확률)
