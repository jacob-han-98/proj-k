## 한 줄 설명
Project K의 워리어 클래스 액티브 스킬, 패시브 스킬, 구현 요구사항을 정의한 문서. 공격적 성향의 고HP 기반 생존형 클래스로 스턴과 출혈 CC기, 자가 강화 버프를 중심으로 구성.

## 숫자/상수/공식
- 비셔스 리프: 1단계 스턴 (10% / 짧은 지속시간)
- 블러드 서지: 1단계 출혈 (10% / 적은 양)
- 블레이드 아크: 2단계 스턴 (50%), 사거리 7m
- 워더링 블레이드: 2단계 출혈 (30% / 중간 양)
- 프라이멀 그랩: 사거리 5m, 2단계 스턴 (50%)
- 버서크: 3초간 무적 + 6초간 광폭화
- 파이널 저지먼트: 3단계 경직 (100%, 1초), 반지름 5m 원형 범위
- 피지컬 인핸스: HP+20
- 스피리트 인핸스: MP+5
- 프리시전: 명중+1
- 웨폰 마스터리: 공격력+1
- 워로드: 10스택마다 HP 소량 회복
- ApplyMaxDamage / 10000 = 100%
