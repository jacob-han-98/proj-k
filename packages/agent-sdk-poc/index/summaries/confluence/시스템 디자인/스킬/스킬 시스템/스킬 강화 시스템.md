# 스킬 강화 시스템 (요약)

> 출처: 시스템 디자인/스킬/스킬 시스템/스킬 강화 시스템
> 원본: packages/confluence-downloader/output/시스템 디자인/스킬/스킬 시스템/스킬 강화 시스템/content.md

## 한 줄 설명
습득한 스킬을 재화와 재료를 소모하여 강화하는 시스템으로, 스킬 등급/레벨별 확률에 따라 성공/실패/단계 하락의 3가지 결과가 발생한다.

## 핵심 용어
- 스킬 강화
- 강화 레벨
- 강화 성공 확률
- 강화 단계 하락 확률
- 강화 유지 확률
- 스킬 등급 (일반, 고급, 희귀, 영웅, 전설)
- 강화 재료
- 모체스킬 (MotherSkill)
- SkillEnchantClass
- SkillEnchantItemClass
- 자동 강화
- 스킬 초월
- 클래스 (가디언, 워리어, 아처, 아바레스트, 매지션, 샤먼)
- 스킬 ID 규칙
- 강화 연출
- 대체 강화 재료
- 목표 단계

## 숫자/상수/공식
- 스킬 레벨 기본값: 0
- 강화 확률 계산식: 10000(만분율) - 강화 성공확률+@ - 강화 단계 하락 확률+@ = 강화 유지 확률
- 강화 유지 확률이 0 미만일 경우: 유지확률 = 0, 강화 성공확률 = 10000 - 강화단계 하락 확률
- 일반 등급: 레벨 1~5 (레벨 5부터 단계 하락 확률 50%)
- 고급 등급: 레벨 1~5 (레벨 5부터 단계 하락 확률 50%)
- 희귀 등급: 레벨 1~5 (레벨 4부터 단계 하락 확률 30%, 레벨 5는 50%)
- 영웅 등급: 레벨 1~5 (레벨 3부터 단계 하락 확률 15%)
- 전설 등급: 레벨 1~5 (레벨 2부터 단계 하락 확률 5%)
- 스킬 초월: 최대 10회 시도 가능
- 클래스 코드: 90(All), 10(가디언), 20(워리어), 30(아처), 40(아바레스트), 50(매지션), 60(샤먼)
- 스킬 ID 예시: 110105(가디언 1번 스킬 5레벨), 210101(아처 1번 스킬 1레벨), 310303(매지션 3번 스킬 3레벨)
- 자동 강화 단계: 5단계씩 표시 (5/10/15 단계로 구분)
- RagdollTypeIndex: 0(default 2500/05), 1(평타 2500/05), 2(발동 5000/30), 3(스킬 8000/45)
