# 스킬 대미지 배율 추가 (요약)

> 출처: 시스템 디자인 / 스킬 시스템
> 원본: packages/confluence-downloader/output/시스템 디자인/스킬/스킬 시스템/스킬 대미지 배율 추가/content.md

## 한 줄 설명
기존 정수 Dice 대미지 방식에 배율(Multiplier) 방식을 추가하여 스킬의 단계 및 성장 방식을 간단하게 관리하기 위한 개선 사항.

## 핵심 용어
- 스킬 배율 (Multiplier)
- 스킬 대미지 (SkillDamage)
- DiceBase
- DiceRoll
- DiceSpot
- DiceCount
- SumAttack
- SumDefence
- FinalDamage
- SkillClass
- 무기 대미지
- 공격자 스탯
- 방어자 스탯
- 스킬 추가 대미지
- 도트 대미지
- MonsterClass
- 도깨비

## 숫자/상수/공식
- SkillDamage = DiceBase + DiceRoll(DiceSpot, DiceCount)
- FinalDamage = (무기 대미지 + 공격자 스탯 + 스킬 대미지) - 방어자 스탯
- [현재] Damage = SumAttack - SumDefence
- [개선] SumAttack = {(무기 대미지 + 공격자 스탯) * 스킬 배율} + 스킬 대미지(Dice)
- 10000 = 100%
- 평타 = 100%
- 예시: 대미지형 스킬 A는 150%, 광역 스킬 B는 40%, 대미지형 스킬 B는 120% + 스킬 대미지 50
