## 한 줄 설명
클래스에 종속된 액티브 스킬을 퀵슬롯에 장착/해제/교체할 수 있는 시스템으로, 12개 슬롯을 슬라이드 또는 확장 형태로 제공한다.

- 채팅창 HIDE
- 투명도 변화 애니메이션

## 숫자/상수/공식
- 12개 슬롯 제공
- 3개 페이지 (0페이지, 1페이지, 2페이지)
- 4개의 액티브 스킬 (게임 초반 획득 기준)
- 쿨타임: 남은 시간 비율만큼 시계 방향으로 딤드 해제
