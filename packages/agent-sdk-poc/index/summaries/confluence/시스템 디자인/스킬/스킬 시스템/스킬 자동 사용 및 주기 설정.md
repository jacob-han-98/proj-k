## 한 줄 설명
스킬의 자동 사용 여부, 사용 주기, 발동 순서를 통합하여 관리하는 기능으로, 플레이어가 메뉴 > 스킬에서 각 스킬마다 자동 사용 상태와 주기를 설정할 수 있다.

## 숫자/상수/공식
- 분 설정 범위: 최소값 0, 최대값 99
- 초 설정 범위: 최소값 0, 최대값 59
- 현재 MP 소모량 = 현재 자동 사용 중인 스킬들의 15초동 평균 사용량의 총합
- 변경 MP 소모량 = 사용 주기를 변경 시킨 해당 스킬의 평균 사용량의 변동값 합산
- + 버튼: 1씩 증가
- - 버튼: 1씩 감소
