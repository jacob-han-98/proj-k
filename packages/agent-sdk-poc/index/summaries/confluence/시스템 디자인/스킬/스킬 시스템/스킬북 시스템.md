## 한 줄 설명
스킬북 아이템을 획득하고 사용하여 스킬을 습득하는 방식을 정의하며, 기존 변신 종속 구조에서 클래스 종속 구조로 변경되는 스킬 시스템의 규칙과 테이블 구조를 명시한다.

## 핵심 용어
- 스킬북 아이템
- 클래스 기반 스킬 구조
- 공용 패시브 스킬
- 변신 전용 스킬
- ConsumeType
- SkillBook (신규 타입)
- ConditionClassType
- ItemBoxId
- 스킬 습득 안내
- 스킬북 아이템 툴팁
- 가디언
- 워리어
- 아처
- 매지션
- 등급 (일반/고급/희귀/영웅/전설/신화)
- NPC 상점
- 퀘스트 보상
- 몬스터 드랍템
- 자동 필터링 기능

## 숫자/상수/공식
- 아이템 ID 규칙: 2,000,000 + 스킬 ID
- 스킬 습득 안내 표시 시간: 약 3초
- 클래스 타입 코드: 0=All, 1=Guardian, 2=Warrior, 3=Archer
- 예시 아이템 ID: 2,000,301 (TripleArrow), 2,003,001 (StormShot), 2,000,501 (Fireball)
- 예시 판매가: 100 골드 (일반), 10,000 골드 (전설)
- MaxStack: 1 (예시 기준)
- ConditionMinLV 예시: 0, 50
- 시간 입력 형식: 24시간 기준 (예: 23:00 = 오후 11시, 00:00 = 자정)
