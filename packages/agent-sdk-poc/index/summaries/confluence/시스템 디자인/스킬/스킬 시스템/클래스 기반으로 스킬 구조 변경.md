## 한 줄 설명
기존 변신 종속 스킬 구조를 클래스 종속 구조로 변경하며, 클래스 전용 스킬·공용 스킬·변신 고유 효과로 구성하는 스킬 시스템 재설계 및 UI/데이터 테이블 변경 사항을 정의한다.

## 숫자/상수/공식
- 공용 스킬 ClassId = "0"
- HUD 스킬 슬롯 자동 등록 순서 = 1 → 4번
- "New" 마크 출력 조건: 스킬 획득 후 15분 이내 AND 스킬창 최초 오픈
- 전설 등급 이상 고유 애니메이션: Idle_P, Idle_B, Run_P, Run_B, Attack_01, Attack_02, Ultimate
- 공통/고유 애니메이션 동일 사양 예시: 재생 시간 1.5초, Hit 타이밍 1.2초
