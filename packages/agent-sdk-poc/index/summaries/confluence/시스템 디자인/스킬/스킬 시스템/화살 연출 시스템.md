## 한 줄 설명
화살 피격 시 화살대가 캐릭터 바디의 소켓에 박히는 연출을 구현하기 위한 시스템으로, 스켈레탈 타입별 소켓 세팅과 적용 규칙을 정의한다.

## 핵심 용어
- 화살 연출 시스템
- 소켓
- 스켈레탈 타입
- 2족보행
- 4족보행
- 피격 방향
- 전면/후면 파라메터
- 화살 메쉬
- 볼트 메쉬
- EmbeddedArrow
- 보스 타입
- 개체별 최대 생성 개수
- 시체 소멸

## 숫자/상수/공식
- 최대 생성 개수: 5개 (예시)
- 생성 후 소멸 시간: 20초 (예시)
- EmbeddedArrow Null: 0 (동작하지 않음, 기본값)
- EmbeddedArrow Arrow: 1 (화살 메쉬 사용)
- EmbeddedArrow Bolt: 2 (볼트 메쉬 사용)
