## 한 줄 설명
대상에게 걸린 특정 ID의 버프를 즉시 제거하는 이펙트 효과의 정의 및 입력 방식을 명시한 문서.

## 핵심 용어
- DeleteBuff
- EffectCategoryEnum
- EffectClass
- BuffID
- BuffStackNum
- BaseValue
- 버프 스택
- 중첩 효과
- 상태 이상
- 동상
- 빙결
- 공격력 증가
- Option
- 스택 개수

## 숫자/상수/공식
- 기본 제거 스택 개수: 1개
- 제거 규칙: 삭제할 스택 개수 > 대상 스택 개수인 경우, 대상 스택 개수만큼만 제거

## 참조 시스템
- 스택(중첩) 효과

## 주요 섹션
- 정의 및 입력 정보
- 기본 규칙
- 테이블 구조
