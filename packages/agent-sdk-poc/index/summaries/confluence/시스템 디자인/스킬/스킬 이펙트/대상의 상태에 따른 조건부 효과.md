## 한 줄 설명
자신 또는 상대의 HP, MP, 거리, 보호막 상태 등을 조건으로 검사하여 조건부 효과를 부여하는 시스템을 정의한다.

## 핵심 용어

- BattleEntity
- HpPercent
- MpPercent
- Distance
- HasBarrier
- Self
- Other
- Operator
- BuffClass
- EffectClass
- Target
- Type
- Value
- Comparer
- 조건부 효과
- 보호막

## 숫자/상수/공식
- 10000 => 100% (HP/MP 퍼센트 변환)
- 100 => 1m (거리 변환)
- 0 = 보호막 없음, 1 = 보호막 있음
- HP 70% 이상 (예시 조건)
- 6M 이내 (예시 거리)
- 100% 확률 (예시 발동 확률)
- 추가 피해 +1000 (예시 수치)
