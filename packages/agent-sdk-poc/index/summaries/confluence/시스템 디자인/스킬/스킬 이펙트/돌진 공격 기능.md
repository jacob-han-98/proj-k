# 돌진 공격 기능 (요약)

> 출처: 시스템 디자인 / 스킬 / 스킬 이펙트 / 돌진 공격 기능
> 원본: packages/confluence-downloader/output/시스템 디자인/스킬/스킬 이펙트/돌진 공격 기능/content.md

## 한 줄 설명
워리어의 돌진 스킬로, 시전자가 대상 앞으로 순간 이동 후 공격하는 기능의 동작 규칙, 조건, 우선순위, 테이블 구조를 정의한다.

## 핵심 용어
- 돌진 스킬
- 순간 이동
- CombatZoneType
- 배경 프랍
- 실린더
- 직선상
- 타겟
- 시전자
- 장애물 체크
- 도착 지점 우선 순위
- SkillCategoryEnum
- Dash
- Options 컬럼
- 카메라 이동 딜레이
- 시전자와 타겟 간 거리값
- 충돌체
- TextKey
- Battle_CanNotTarget
- Battle_NotExistTarget
- 텔레포트
- 로그아웃

## 숫자/상수/공식
- CombatZoneType: Normal, Safety Chaotic, Town (Town 제외 전투 가능)
- 카메라 이동 딜레이: 공란 또는 0 = 딜레이 없음, 1000 = 1초
- 도착 위치 기준 거리값: 공란 또는 0 = 실린더와 실린더가 맞닿는 위치, 1 이상 = 해당 입력값만큼 거리 발생

- 이동 후 도착 지점 우선 순위
