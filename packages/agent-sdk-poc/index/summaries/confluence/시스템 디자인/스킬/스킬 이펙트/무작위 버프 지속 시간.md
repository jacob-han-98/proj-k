## 한 줄 설명
버프의 지속시간을 최소값과 최대값 사이의 무작위 시간으로 적용하는 기능으로, 지정된 간격으로 선택 가능한 값들을 결정한다.

## 핵심 용어
- 버프 (Buff)
- 지속시간 (Duration)
- 최소 지속시간
- 최대 지속시간 (MaxDuration)
- 무작위 지속시간 간격 (RandomDurationInterval)
- BuffClass
- 침묵 (Silence)
- 랜덤 미적용
- 간격 값

## 숫자/상수/공식
- 최소 지속시간 2초, 최대 지속시간 3초, 무작위 간격 0.1 → 2, 2.1, 2.2, ..., 2.9, 3.0 중 선택
- 최소 지속시간 2초, 최대 지속시간 3초, 무작위 간격 0.5 → 2, 2.5, 3.0 중 선택
- 최소 지속시간 2초, 최대 지속시간 3초, 무작위 간격 1 → 2, 3 중 선택
- 최소 지속시간 2초, 최대 지속시간 3초, 무작위 간격 0.4 → 2, 2.4, 2.8, 3 중 선택
- 침묵 시간 2초~6초, 1초 간격 → 2, 3, 4, 5, 6 중 선택
