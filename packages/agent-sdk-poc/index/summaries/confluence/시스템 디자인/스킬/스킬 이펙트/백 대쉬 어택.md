## 숫자/상수/공식
- 이동 거리 예시: 300cm, 500cm
- 이동 시간 예시: 0.3초간 5m 이동
- 이동시간(초) = 이동시간(초) / (스킬시전속도 + 1)
- 스킬 시전 속도 0% 기본값: 이동 시간 0.3초
- 스킬 시전 속도 50% 예시: 0.3 / (0.5 + 1) = 0.2초
- 스킬 시전 속도 100% 예시: 0.3 / (1 + 1) = 0.15초
