## 한 줄 설명
대상의 특정 버프 존재 여부에 따라 조건부 효과를 발동하는 시스템으로, 버프 스택 수에 비례하여 효과를 강화하는 메커니즘을 정의한다.

- EffectCategoryEnum
- BonusEnum

- ClassId
- ValuePerBuffStack
- MaxStack
- ConditionProb
- BuffID
- 발키리 스피릿
- 사전 강화 버프
- 스택 비례량

## 숫자/상수/공식
- MaxStack >= 1 (스택 체크 가능 조건)
- MaxStack = 0 (서버 에러 발생)
- 예시: 20스택당 1초 증가
