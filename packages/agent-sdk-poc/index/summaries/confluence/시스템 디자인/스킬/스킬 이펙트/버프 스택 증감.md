## 한 줄 설명
스킬 사용 시 특정 버프의 스택을 증가 또는 감소시키는 이펙트 카테고리의 정의 및 규칙을 명시한 문서.

## 핵심 용어
- AddBuffStack
- EffectCategoryEnum
- BuffID
- BuffStackNum
- 스택(중첩) 효과
- 버프
- 디버프
- EffectClass
- BaseValue
- Value
- 트랜스 버프
- 출혈 디버프

## 숫자/상수/공식
- 스택 최소값: 0 이하로 떨어지지 않음
- 음수(-) 값으로 스택 제거 가능
- 스택이 없는 버프도 -1로 제거 가능
