## 한 줄 설명
기존 스탯에 비율을 곱하여 증감시키는 이펙트 기능으로, 절대값이 아닌 비율 방식의 스탯 조정을 정의한다.

## 핵심 용어
- 스탯 비율 증감
- EffectClass
- EffectCategory
- StatIncreaseRate
- BonusEnum
- Normal 카테고리
- Effect1~5
- Value1~5
- 절대값
- 비율 값
- 소수점 이하 버림 처리
- 회피
- 공격력

## 숫자/상수/공식
- 기존 스탯 + 기존스탯 * 비율 = 최종 값
- -10000 = -100%
- 1000 = 10%
- 0 = 동작 하지 않음
