## 한 줄 설명
스킬 효과로 인해 조건부로 상승하는 스탯이 의도한 범위를 벗어나지 않도록 상한선을 설정하는 기능 정의.

## 핵심 용어

- Max 컬럼
- BonusEnum
- 상한선
- 스탯 효과
- 스탯 비율 보너스
- 스택 보너스
- Effect
- Category
- 상한선값
- 에러 처리

## 숫자/상수/공식
- HP n마다 공격력 n 증가
- n 스택마다 스탯 n 증가
- HP 500마다 공격력 1 증가, 공격력 증가 Limit 5
- HP 5000일 때 공격력+10 → 최종 공격력+5 (상한선 적용)
