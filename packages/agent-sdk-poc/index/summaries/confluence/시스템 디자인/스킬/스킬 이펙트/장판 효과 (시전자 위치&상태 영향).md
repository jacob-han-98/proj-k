## 한 줄 설명
시전자의 위치 및 상태(사망, 범위 이탈)에 영향을 받는 장판 효과의 정의 및 구현 규칙을 명시한 문서. 기존 PulseAreaAttack에 신규 기능을 추가하는 방식으로 개선.

## 핵심 용어

- 장판 효과
- 시전자 (캐스터)
- 장판 제거 조건
- 우호/적대 판정
- BuffId
- SkillCategoryEnum
- WorldViewRange
- AreaShape
- AreaSize
- AreaStandard
- AreaLocation
- AreaFxResource
- Options 컬럼
- Dice 정보
- Duration
- 토템 스킬
- 샤먼
- 버프/디버프 효과
- 틱 (Tick)

## 숫자/상수/공식
- 틱 주기: 약 0.5초 권장
- 버프 적용/해제 오차: 최대 0.5초
- BuffId Duration (범위 진입/이탈 방식): -1 (무제한)
- 틱당 시간 ≤ 최대 출력 시간 (필수 조건)
