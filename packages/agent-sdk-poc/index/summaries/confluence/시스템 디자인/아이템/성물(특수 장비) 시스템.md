## 한 줄 설명
Project K의 특수 장비 시스템으로, 오딘/뱀피르와 유사한 방식의 성물 파츠 착용, 강화, 프리셋 관리 및 UI 개편을 정의한다.

## 숫자/상수/공식
- 최초 4파츠 개방, 최대 8파츠까지 제공
- +1강 마다 명중 +1 증가
- +4, 8, 12강 마다 확률형 능력치 추가
- +2강부터 파괴 위험 있음
- EquipmentNotifyLifeTime = 10초
- 장착/해제 재사용 대기 시간 = 0.1초
- 프리셋 변경 재사용 대기 시간 = 0.5초
- 최대 2개의 프리셋 기본 제공
- 슬롯 표시/숨김 설정값 = HolyRelicMax
