## 한 줄 설명
강화 실패로 장비가 파괴되었을 때, 해당 장비의 분해 재료를 환원해주는 시스템으로, 분해 로직을 활용하여 재료를 반환한다.

## 핵심 용어
- 강화 실패
- 장비 파괴
- 분해 재료
- 재료 환원
- ItemEquipClass
- CanDismantle
- 분해 가능 장비
- 분해 불가 아이템
- DismantleReward
- DismantleProbabilityReward
- 분해 테이블
- 강화 파괴 연출
- 분해 결과 UI

## 숫자/상수/공식
- CanDismantle = TRUE (분해 가능 조건)

## 참조 시스템
- 분해 시스템 (DismantleReward / DismantleProbabilityReward 로직)

## 주요 섹션
- 개요
- 규칙
- 적용할 대상 장비
- 장비 파괴 시
- 분해 결과 안내 시점
