## 한 줄 설명
장비 아이템의 기본 스펙을 올려주는 게임 내 기본 성장축으로, 강화 재료와 골드를 소비하여 확률 기반의 성공/실패 여부에 따라 스탯을 상승시키거나 장비를 소멸시키는 시스템.

- 일반 강화주문서

- 단일 강화
- 다중 강화
- 컬렉션 강화 단계 표시

## 숫자/상수/공식
- SuccessProb = 100% (안전 강화 구간 정의)
- BlessedSuccessProb01~03 합계 = 100%
- 소수점 둘째자리까지 사용 (확률 표기)
- 강화 단계 표기: +{강화단계} {ItemName}
- 테이블 기입 규칙: {아이템 ID}+{강화단계}
- 로그 보존 기간: 최대 3개월
