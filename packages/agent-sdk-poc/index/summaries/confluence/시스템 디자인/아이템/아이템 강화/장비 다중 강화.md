## 한 줄 설명
1개 이상의 장비 아이템을 목표 강화 단계까지 일괄 자동으로 강화하는 시스템으로, 대량 획득 아이템의 반복 강화 불편함을 해소한다.

## 핵심 용어
- 장비 다중 강화
- 단일 강화
- 강화 주문서
- 강화 ID
- enchant_id
- 목표 강화 단계
- 강화 재료
- 축복받은 재료
- 거래 가능
- 캐릭터 귀속
- 슬롯
- Group ID
- 강화 연출
- 분해 결과
- 강화 취소
- 동일 아이템 일괄 등록
- 확률 정보
- EnchantWeapon
- BlessedEnchantWeapon
- CursedEnchant

## 숫자/상수/공식
- 최대 등록 장비 수: 50개 (ContentSetting > MultiEnchantMaxCount = 50)
- 최대 강화 단계: +15
- 축복받은 재료 강화 단계 증가: +1~+3
- 강화 재료 계산 예시: +7강 목표, +0, +1, +2 아이템 등록 시 (7-0)+(7-1)+(7-2) = 20장 필요
