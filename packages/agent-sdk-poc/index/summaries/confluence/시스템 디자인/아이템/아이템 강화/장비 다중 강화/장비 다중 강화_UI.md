## 한 줄 설명
여러 장비를 한꺼번에 강화할 수 있는 다중 강화 모드의 UI 설계 문서로, 단일 강화와 구분되는 탭 메뉴, 장비 선택 기능, 목표 강화 단계까지의 자동 진행 방식을 정의한다.
