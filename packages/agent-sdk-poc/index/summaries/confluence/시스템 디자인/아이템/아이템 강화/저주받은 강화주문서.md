## 한 줄 설명
장비 아이템의 강화 단계를 100% 확률로 -1 하락시키는 강화주문서로, 안전 강화 단계에서 고강을 띄우는 플레이를 유도하는 시스템.

## 숫자/상수/공식
- 강화 단계 변화: -1 (100% 확률)
- 등록 가능 강화 단계: +1 이상 ~ MAX 레벨
- 0강: 등록 불가
- CursedEnchantGoldPrice: int 타입 상수 (0 가능)
