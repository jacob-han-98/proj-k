# 아이템 분해_UI (요약)

> 출처: 시스템 디자인 / 아이템 분해
> 원본: packages/confluence-downloader/output/시스템 디자인/아이템/아이템 분해/아이템 분해_UI/content.md

## 한 줄 설명
아이템을 선택하여 등록하고 분해 결과(확정/확률 보상)를 확인한 후 최종 분해를 진행하는 UI 시스템의 화면 구성 및 동작 명세.

## 핵심 용어
- 개별 분해 버튼
- 일괄 분해 버튼
- 분해 모드
- 분해 목록 출력 공간
- 분해 모드 탭
- 등급 필터
- 선택된 아이템 수량
- 분해 버튼
- 예상 분해 결과 UI
- 확정 보상
- 확률 보상
- DismantleFix
- DismantleRandom
- 최종 분해 결과 UI
- 분해 성공
- 획득 보상 리스트
- 인벤토리 슬롯
- 잠금 상태 아이템
- 일반/고급/희귀 등급

## 숫자/상수/공식
- 확정 보상 리스트: 최대 3.5개 출력
- 확률 보상 리스트: Min, Max 값 표시 (Min~Max 또는 n 형식)
- 확률 보상 계산: Prob < 100% 시 "0~(분해 개수*AmountMax)", Prob = 100% 시 "1~(분해 개수*AmountMax)"
- 획득 보상 리스트: 최대 6.5개 출력
- 결과 화면 닫기 불가 시간: 최초 2초
