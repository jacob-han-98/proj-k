## 한 줄 설명
아이템을 분해하여 확정 보상과 확률 보상을 획득하는 시스템의 규칙, 필터링 조건, 테이블 구조를 정의한 문서.

## 핵심 용어
- CanDismantle
- 분해 모드
- 가방 모드
- 확정 보상
- 랜덤 보상
- DismantleRewardId
- DismantleProbabilityRewardId
- ItemEquipClass
- DismantleReward
- DismantleProbabilityReward
- RewardType
- ItemId
- Amount
- Prob
- 분해 가능 대상
- 자동 선택 조건
- 분해 성공 확률
- 데이터 연결
- 자동 그룹화
- AmountMin
- AmountMax

## 숫자/상수/공식
- 1,000,000 = 100% (확률 기준)
- +0강 (자동 선택 조건)
- Default Amount: 1
- Default Prob: 1,000,000
- Default AmountMin: 1
- Default AmountMax: 1

## 참조 시스템
- 아이템 분해_UI

- 전체 플로우
- 분해 모드
- 세부 규칙
- 분해 가능 대상
- 분해 모드 시 자동으로 선택될 아이템 조건
- 분해 성공 확률
- 분해 비용 및 기타
- 테이블 구조
