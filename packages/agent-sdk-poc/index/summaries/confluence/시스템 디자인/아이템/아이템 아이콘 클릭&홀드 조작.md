## 한 줄 설명
아이템 아이콘의 클릭과 홀드(롱프레스) 조작을 구분하여, 홀드 시 아이템 정보 툴팁을 표시하는 UX 기능을 정의한다.

## 핵심 용어
- 아이템 아이콘
- 클릭&홀드 조작
- 롱프레스
- 아이템 정보 툴팁
- 게이지바
- IconHoldStartTime
- IconHoldTime
- DA_ClientDataAsset
- 인벤토리
- 강화
- 컬렉션
- 거래소
- BM 상점
- Secondary 기능
- 오작동 판단

## 숫자/상수/공식
- IconHoldStartTime = 400~500 (ms) = 0.4~0.5초
- IconHoldTime = 500 (ms)
- 게이지 % = 실제 홀드 중인 시간 / IconHoldTime
