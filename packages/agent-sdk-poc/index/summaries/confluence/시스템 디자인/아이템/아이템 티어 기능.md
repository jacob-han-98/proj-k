## 한 줄 설명
장비 아이템의 등급 내에서 티어를 나누어 유저에게 장비의 급을 알리고 정렬하기 위한 기능.

## 숫자/상수/공식
- Tier = 0 또는 공란: 티어 없음 (T1보다 낮음)
- domain = cs / type = int32
