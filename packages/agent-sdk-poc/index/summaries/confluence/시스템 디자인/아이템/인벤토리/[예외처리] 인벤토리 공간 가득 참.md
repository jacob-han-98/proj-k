## 숫자/상수/공식
- 여유 공간 계산식 (분해/상자/제작): (현재 인벤토리 잔여 공간) + (행위로 소멸되는 대상이 차지하던 총 공간 수) ≥ (결과물 획득에 필요한 총 공간 수)
- 여유 공간 계산식 (우편): (현재 인벤토리 남은 공간) ≥ (해당 우편에 들어있는 모든 아이템을 받기 위해 필요한 총 공간 수)
- 능동적 획득 기준: 결과물 1종류당 1개의 추가 빈 슬롯 필요 (중첩 여부 무관)
- 수동적 획득 기준: 빈 슬롯 0칸 상태에서도 기존 스택에 합산 가능 (최대치 미도달 시)
