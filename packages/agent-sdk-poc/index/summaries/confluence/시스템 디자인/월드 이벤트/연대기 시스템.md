# 연대기 시스템 (요약)

> 원본: packages/confluence-downloader/output/시스템 디자인/월드 이벤트/연대기 시스템/content.md

## 한 줄 설명
전체 서버의 타임라인을 기록하고 열람할 수 있는 역사 페이지로, 프리시즌부터 반복되는 시즌별 월드 이벤트 기록과 플레이어 성취를 관리하는 시스템.

## 핵심 용어
- 연대기
- 프리시즌(PreSeason)
- 시즌N
- 월드 이벤트
- 공성전
- 인터서버 전장
- 명예의 전당
- 기록(실록)
- 서버 파워
- 이벤트 타입
- 전투력 성장
- 스킬 강화
- 장비 아이템 강화
- 정령의 탑 도전 단계
- 월드 보스 처치 경쟁
- 필드 보스 처치 경쟁
- 이벤트 몬스터 처치 경쟁
- ChronologyID
- Chronology 테이블
- ChronoloyReward 테이블

## 숫자/상수/공식
- 컨텐츠 해방 캐릭터 레벨 조건: Lv.20
- 정렬 규칙 적용 시점: 3주차부터
- 스크롤 이동 단위: 1주차씩
- 서버 파워 계산: 1등부터 100등까지 서버별 순위/가중치 합산 후 비율 표시

## 주요 섹션
- 개요
- 규칙 (기본, 연대기 구성, 연대기 기록)
- UI (진입 방식, 초기 상태, 스크롤 이동, 선택 상태, 시즌 변경, 이벤트 항목 상태, 최근 기록 이동, 상세 정보)
- 테이블 및 데이터 구조 (테이블 변경, 신규 테이블 추가)
- 텍스트 키
