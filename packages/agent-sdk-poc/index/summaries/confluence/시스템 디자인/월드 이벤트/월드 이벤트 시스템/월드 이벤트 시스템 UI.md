## 숫자/상수/공식
- 최대 랭킹 집계 순위: 100위
- 최종 순위 결정: 이벤트 종료 24시간 전
- 남은 시간 1일 이상: [00일 00시간] 표기
- 남은 시간 1일 미만: [HH:MM:SS] 표기
- 순위 구간별 보상 최대: 5개 아이템
- 랭킹 1~3위: 순위 표시 강조
