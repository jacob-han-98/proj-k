## 숫자/상수/공식
- 하루 최대 이벤트 개수: 10개
- 이벤트 시작/종료 기준: GMT+9 자정(0시)
- 상점 종료 시기: 이벤트 종료 후 2~3일 이후
- 상점 이용 불가 기간: 이벤트 종료 후 최소 하루 이상
- 초기 서버 그룹 최소 개수: 3~5개 이상
- 공성전 진입 전 분기 구간: 1~2회
