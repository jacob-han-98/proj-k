## 한 줄 설명
정해진 기간 동안 지정된 이벤트들을 활성화시키는 기능으로, 시작 시간, 적용 서버, 이벤트 기간, 진행 단위를 설정하여 관리한다.

## 숫자/상수/공식
- 순위 1등: 가중치 100, 총합 100
- 순위 2등: 가중치 50, 총합 50
- 순위 3등: 가중치 30, 총합 30
- 순위 4~10등: 가중치 5, 총합 35
- 순위 11~30등: 가중치 3, 총합 60
- 순위 31~60등: 가중치 2, 총합 60
- 순위 61~100등: 가중치 1, 총합 40
- 순위 기본 점수: 1점
- 이벤트 분기 기준 예시: 50점 이상/미만
- 이벤트 목록 유지 시간: 7일(기본값)
- 공성전 진입 전 분기 구간: 1~2회
