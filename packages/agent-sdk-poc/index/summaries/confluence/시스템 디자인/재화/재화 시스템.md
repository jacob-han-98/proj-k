## 한 줄 설명
Project K에서 사용하는 재화의 종류, 귀속 기준, 유료/무료 구분, 저장 구조 및 UI 표현 방식을 정의한 시스템 문서.

## 핵심 용어
- 유료 재화
- 무료 재화
- Diamond (유료 재화, BM 구매)
- FreeDiamond (유료 재화, 컨텐츠 보상)
- Gold (인게임 기본 재화)
- 캐릭터 귀속
- 서버 귀속
- 길드 귀속
- CurrencyClass
- CurrencyEnum
- DbEntityEnum
- RewardTypeEnum
- Mileage
- Coin
- GuildCoin
- GuildFund
- MissionCoin
- InfiniteCoin
- BattleCoin
- HonorCoin
- 유상/무상 재화 분리
- 재화 사용 순서

## 숫자/상수/공식
- FreeDiamond → Diamond 순으로 사용 (재화 소모 우선순위)
- CurrencyEnum 값: Gold(1), Diamond(2), FreeDiamond(3), Mileage(4), Coin(5), GuildCoin(6), GuildFund(7), MissionCoin(8), InfiniteCoin(9), BattleCoin(10), HonorCoin(11)
- DbEntityEnum 값: Character(1), Account(2), Guild(3)
