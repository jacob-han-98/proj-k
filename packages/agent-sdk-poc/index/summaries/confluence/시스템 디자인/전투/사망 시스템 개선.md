## 한 줄 설명
기존 사망 패널티(경험치 손실)를 확장하여 장비 손실 및 스탯 패널티를 추가하고, 성향치에 따라 차등 적용하며, 사망 연출을 강화하는 시스템 개선 사항을 정의한다.

## 숫자/상수/공식
- 경험치 손실률: 5% (모든 성향 공통)
- 장비 손실 개수: 최소 1개~최대 6개
- 손실 장비 목록 최대 보유: 20개
- 경험치 손실 기록 최대 보유: 10개
- 사망 UI 손실 리스트 최대 출력: 8개 (경험치 1개 + 장비 최대 6개 + 디버프 1개)
- 무료 화폐 비용 공식: 기본 감소량 × (장비 등급 계수) × (파츠 계수) × (현재 강화 단계 +1)
- 유료 화폐 비용 공식: "무료 화폐 비용" / 전용 계수 (소숫점 이하 버림, 1 미만 시 1로 보정)
- DB 등록 후 24시간 이후부터 유료 화폐로만 복구 가능
- 손실 확률 최대: 100%
- 손실 개수 최대: 6개
