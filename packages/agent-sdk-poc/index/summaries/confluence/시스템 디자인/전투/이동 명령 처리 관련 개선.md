# 이동 명령 처리 관련 개선 (요약)

## 한 줄 설명
PC의 이동 조작 및 자동 이동 시스템의 베이스 규칙을 정의하고, 현재 빌드(250709)에서 발견된 어색한 이동 처리 동작을 개선하기 위한 방향을 제시하는 문서.

## 핵심 용어
- 자동 이동
- Quest 모드
- Auto 모드
- 직접 제어 이동
- 방향키 WASD
- 퀵이동
- 월드맵 자동 이동
- NPC 더블 클릭
- 마우스 클릭 이동
- 가상 키패드
- 슬라이딩 이동
- 네비메쉬
- 군중 제어(C.C)
- 시스템 메시지
- 이동 명령
- 이동 가능 여부 체크
- PC 상태
- 목적 지점 유효 상태
- 최신 입력 조작 우선시
- 이동 중지

## 숫자/상수/공식
- SystemMsg ID 9,003: AutoPlay_CancelMove (자동 이동 종료 메시지)
- 빌드 기준: 250709

## 주요 섹션
- 기획 의도 및 목적
- 현재 빌드 내 개선 필요점
- (중요도 중) 특정 자동 이동 진행 중 새로운 이동 관련 처리
- (중요도 중) 자동 이동 비활성화 후 PC 이동이 지속되는 건
- (중요도 하) NPC와 상호 작용 UI 호출 중 PC 이동이 가능한 건
- 참고용) 기획적 방향성
- 이동 조작의 입력과 실행 처리
- 이동 진행 중 새로운 이동 조작 발생 시
- 경우에 따른 베이스 규칙 처리 예시
