## 한 줄 설명
캐릭터의 HP가 설정한 비율 이하로 내려갈 시 자동으로 HP 물약을 사용해주는 PC 전용 편의 기능으로, 최대 3종의 물약을 등록하고 교체 방식을 설정할 수 있다.

## 핵심 용어
- 자동 물약 사용 시스템
- HP 물약
- 소형 / 중형 / 대형
- HudHpPotion01, HudHpPotion02, HudHpPotion03
- HudHpPotionCondition
- HpPotionGlobalCoolTime
- ContentSetting
- 교체 방식
- 높은 등급
- 낮은 등급
- 교체 안함
- PVP 모드
- 절전 모드
- 서포트 모드
- 글로벌 쿨타임
- 물약 회복률 증가
- 인벤토리
- HUD

## 숫자/상수/공식
- HudHpPotionCondition 기본값: 85%
- HpPotionGlobalCoolTime 기본값: 85% (입력값 1000 = 1초)
- HP 비율 설정 최소값: 0
- HP 비율 설정 최대값: 100
- 자동 등록 가능 최대 물약 개수: 3개
- PVP 시 HP 비율 기본값: 100% (On)
