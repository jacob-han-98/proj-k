# 자동 전투 시스템 (요약)

## 한 줄 설명
일정 범위 내의 몬스터를 자동으로 타게팅하여 연속성 있게 전투를 진행하는 PvE 기능으로, Auto 버튼, 일반 공격 버튼, 퀘스트 UI를 통해 활성화된다.

## 핵심 용어
- 자동 전투 (Auto 전투)
- 일반 공격
- 퀘스트 자동 전투
- 타게팅
- 탐색 범위
- 회귀 기능
- 매너 사냥
- 몬스터 자동 반격
- HUD 타겟 정보
- 네임 플레이트
- 탐색 우선 순위
- 자동 전투 영역 중점
- 수동 이동
- 수동 타겟팅
- 인터렉션
- FX 연출
- 환경설정 옵션
- 서포트 모드
- 파티원
- 길드원

## 숫자/상수/공식
- 기본 탐색 범위: 20m (DefaultGameData - AutoBattleSearchRaidus 참조)
- 환경설정 범위: 5~40m
- 대상 탐색 주기: 1초마다 (범위 내 몬스터 없을 경우)
- 공격 불가 위치 판정 시간: 10초
- 자동 전투 범위 연출 높이: 약 50cm
