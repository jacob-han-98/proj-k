## 한 줄 설명
PvE 몬스터의 피격 연출 재생 여부와 확률을 제어하는 기능으로, 클라이언트 모션 연출만 영향을 받으며 서버 판정은 변하지 않는다.

## 핵심 용어
- PvE 몬스터
- NPC
- 피격 연출
- 피격 모션
- 연출 (클라이언트)
- 서버 판정
- DisableHitReaction
- HitReactionRate
- idle
- Combat-idle
- 기본 공격
- 평타
- 보스
- 레이드 몹
- 특수 케이스
- 네임드 몬스터
- 일반 몬스터

## 숫자/상수/공식
- DisableHitReaction 기본값: 0
- HitReactionRate 기본값: 50
- HitReactionRate 범위: 0~100%
- DisableHitReaction 값: 0 (피격 연출 활성화), 1 (피격 연출 비활성화)
