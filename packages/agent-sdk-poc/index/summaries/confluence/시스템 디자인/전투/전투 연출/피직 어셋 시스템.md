# 피직 어셋 시스템 (요약)

> 출처: 시스템 디자인 / 전투 / 전투 연출 / 피직 어셋 시스템
> 원본: packages/confluence-downloader/output/시스템 디자인/전투/전투 연출/피직 어셋 시스템/content.md

## 한 줄 설명
근거리 공격과 원거리 공격의 피격 지점을 분리하여 대형~특대형 몬스터의 타격 연출을 자연스럽게 처리하는 시스템.

## 핵심 용어
- 피직 어셋
- 실린더
- 원거리 타격 소켓
- 컬리전
- 발사체
- FX_Center
- FX_Projectile
- Hit Particle Info
- 근거리 공격
- 원거리 공격
- 몬스터 크기 규격
- DetailHit
- ProjectileDetailHit
- 몽타주
- 로컬 액터

## 숫자/상수/공식
- 소켓 종류: 2종류 (FX_Center / FX_Projectile)
- 동작 우선 순위: FX_Projectile > FX_Center > 로컬 액터
- 각 소켓 동시 적용: 1개만 가능
- DetailHit 기본값: FALSE (피직 어셋 기능 Off)
- DetailHit TRUE: 피직 어셋 기능 및 근거리 타격 예외 처리 기능 On

## 참조 시스템
- 몬스터 크기 규격
