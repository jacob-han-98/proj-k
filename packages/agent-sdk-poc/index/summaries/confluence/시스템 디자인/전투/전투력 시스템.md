# 전투력 시스템 (요약)

> 출처: 시스템 디자인 / 전투 / 전투력 시스템
> 원본: packages/confluence-downloader/output/시스템 디자인/전투/전투력 시스템/content.md

## 한 줄 설명
플레이어의 스탯을 가중치 기반으로 계산하여 전투력을 산출하고, 이를 통해 컨텐츠 접근 제어 및 성장 동기 부여를 하는 시스템의 기획 및 개선 방향을 정의한다.

## 핵심 용어
- 전투력
- 1차 스탯
- 2차 스탯
- 가중치
- 클래스 필터링
- 스탯 환산
- 전투 효과
- 클래스
- 궁사
- MeleeAttack
- 게임플레이 직관성
- 전투 로그 분석
- 라이브 서비스
- 패치
- 컨텐츠 접근 제어
- 성장 동기 부여
- 스탯 상승

## 숫자/상수/공식
- 전투력 산출 공식: 사전 정의된 스탯과 해당 스탯의 가중치를 곱하여 합산
- 클래스 필터링 시 가중치: 0 처리

## 참조 시스템
- PK_전투력 시스템.xlsx

## 주요 섹션
- 기획 의도 및 목적
- 전투력 정의
- 전투력 산출 공식
- 스탯 분류
- 클래스 필터링
- 기존 기획 중 유지할 내용
- 기존 기획 대비 추가/변경할 내용
- 기획팀 참고 내용
- 우려되는 점
