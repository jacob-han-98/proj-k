## 숫자/상수/공식
- Fade in: 0.3초 (변경)
- 유지: 3초 (변경)
- Fade out: 0.3초 (변경)
- 페이지 전환 시 각 페이지 출력 시간: 3초
- 스탯 증감 표시 최대 자릿수: 8자리
- 페이지 전환 조건: 스탯 증감 내용 3개 초과 시
