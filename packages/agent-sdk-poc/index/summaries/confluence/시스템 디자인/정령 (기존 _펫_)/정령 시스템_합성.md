## 한 줄 설명
동일 등급의 정령 4개를 동일 또는 상위 등급의 정령 1개로 교환하는 합성 기능을 정의한 시스템 설계 문서.

## 핵심 용어
- 정령 (Pet)
- 합성 (Compose)
- 강화 (Enchant)
- 확률 강화
- 안전 강화
- 변신_합성
- 등급 (Grade)
- 재료 (Material)
- PetComposeList
- PetClassId
- UseGrade
- 소환 (Summon)
- 흐름도
- 시스템 메시지
- 네트워크 이슈
- 복구 불가
- 단일 소환/합성
- 복수 소환/합성
- Exp
- 성공 확률
- 하수구

## 숫자/상수/공식
- 동일 등급 4장당 1회 합성 가능
- 4장 미만인 경우 합성 불가
- 합성 성공 확률: 100%
- Prob 입력값: 10,000 = 100%
- 동일 Id의 Prob 합계: 10,000(=100%)이어야 함
