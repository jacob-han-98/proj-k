## 숫자/상수/공식
- 총 33종 메시
- 지팡이 10종 (전설 1, 영웅 4, 희귀 3, 고급 1, 일반 1)
- 오브 10종 (전설 1, 영웅 4, 희귀 3, 고급 1, 일반 1)
- 마도서&스크롤 9종 (전설 1, 영웅 3, 희귀 3, 고급 1, 일반 1)
- 반지 4종 (전설 1, 영웅 1, 희귀 2, 고급 0, 일반 0)
- Idle 재생 시간: 1~2초
- Run 재생 시간: 1초
