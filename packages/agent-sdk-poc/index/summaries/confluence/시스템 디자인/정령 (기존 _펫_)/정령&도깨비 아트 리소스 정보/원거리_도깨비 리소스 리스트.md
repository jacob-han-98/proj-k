## 한 줄 설명
원거리 도깨비 캐릭터의 애니메이션, 스킬 이펙트, 아이콘 리소스 제작을 위한 상세 명세서. 탱딜 균형의 서포터 역할로 어그로 분산과 광역 기절 기능을 수행한다.
