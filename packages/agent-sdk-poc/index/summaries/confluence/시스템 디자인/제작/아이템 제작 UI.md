## 숫자/상수/공식
- 즐겨찾기 수량 제한: 20개 (ItemCraft_Fav = 20)
- 제작식 링크 최대 뎁스: 3단계
- 제작 수량 기본값: 1개
- 제작 수량 최소값: 1개
