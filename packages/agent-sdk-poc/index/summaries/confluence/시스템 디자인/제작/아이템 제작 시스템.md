# 아이템 제작 시스템 (요약)

> 출처: 시스템 디자인/제작/아이템 제작 시스템
> 원본: packages/confluence-downloader/output/시스템 디자인/제작/아이템 제작 시스템/content.md

## 한 줄 설명
인벤토리 및 창고의 재료와 재화를 소모하여 새로운 아이템을 제작하는 시스템으로, 아이템 드랍에 드는 장기간의 노력을 가시적으로 분할하며 제작 경험치/레벨 없이 재료만 있으면 공평하게 진행 가능하다.

## 숫자/상수/공식
- 최대 8종류의 아이템 투입 가능
- 최대 2종류의 재화 투입 가능
- 최대 3종의 반환 재료
- 즐겨찾기 최대 20개
- 10000 = 100% (확률 단위)
- 대성공 확률 = 성공 확률의 n% (예: 성공 80%, 대성공 10% 세팅 시 실제 대성공 확률 8%)
- 재료 소모 우선순위: ① 인벤토리 CanStorage=FALSE → ② 인벤토리 CanAuction=FALSE → ③ 창고 CanAuction=FALSE → ④ 인벤토리 CanStorage=TRUE → ⑤ 창고 CanStorage=TRUE → ⑥ 인벤토리 CanAuction=TRUE → ⑦ 창고 CanAuction=TRUE
