# 클래스 디자인 - 아발리스터 (요약)

> 출처: 시스템 디자인/클래스
> 원본: packages/confluence-downloader/output/시스템 디자인/클래스/클래스 디자인 - 아발리스터/content.md

## 한 줄 설명
시즈 모드 공격 시 순간 화력이 강한 원거리 누킹 딜러 클래스로, 체력/보호막 대상에 특화되고 회피/방어력으로 생존력을 보충하는 아발리스터의 애니메이션, 이펙트, 스킬 연출을 정의한다.

## 숫자/상수/공식
- Idle 모션 재생 시간: 1.5초
- 기본 공격/스킬 애니메이션 재생 시간: 1.7초
- 전설 등급 달리기 애니메이션 재생 시간: 0.87초
- 스모크 파이어 범위(반지름): 4M
- PvE 조작 난이도: 보통
- PvP 조작 난이도: 어려움
