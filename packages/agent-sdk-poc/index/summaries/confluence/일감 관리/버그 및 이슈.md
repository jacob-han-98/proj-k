## 숫자/상수/공식
- PlayerResawnDelayTime = 30초
- 경로 찾기 실패 지점: 계단 초입, 아비시 대각선 옆
- 퀘스트 목표 수량 오류 (5마리 이상 진행 불가)
- Rectangle 타입 범위 설정: 200, 500 => 4m, 10m (실제 10m로 표시)
- 무릎 화살 스케일: 2.0
- 호은 발생 확률: 약 92%
