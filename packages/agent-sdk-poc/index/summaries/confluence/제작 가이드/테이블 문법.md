# 테이블 문법 (요약)

> 출처: 제작 가이드 / 테이블 문법
> 원본: packages/confluence-downloader/output/제작 가이드/테이블 문법/content.md

## 한 줄 설명
Project K 시스템 기획 문서에서 사용되는 다양한 테이블 문법과 값 입력 방식을 정의한 메뉴얼.

## 핵심 용어
- 테이블 문법
- 장비 아이템 강화
- EnchantLv
- ItemId
- 그룹
- Collection
- EffectClass
- BonusEnum
- 스탯 상한선
- Max 컬럼
- 값 입력 방식

## 숫자/상수/공식
- {ItemId}+{EnchantLv} (예: 1000+7)
- g{ItemId} (그룹 표기법, 예: g1000)
- {BonusEnum}={상한선 값} (예: AllAttack=5, MaxHp=500)

## 참조 시스템
- 장비 강화 시스템
- PK_Collection 테이블
- PK_EffectClass 테이블

## 주요 섹션
- 장비 아이템 강화
- 그룹
- 스킬별 스탯 효과의 상한선
