## 한 줄 설명
Project K의 월드맵 시스템에서 자동 이동, 미발견 지역 표시, 확대/축소 조작감, 드랍 아이템 정렬, 국가명 표시, 자동 이동 메시지, 아이콘 비율 조정, 신규 지역 추가 등 8가지 개선 항목을 정의한다.

## 핵심 용어
- 자동 이동
- 텔레포트
- 텔레포트 볼륨
- 미발견 지역
- Territory 구역
- 발견 지역
- 월드맵
- 확대/축소
- 줌인/줌아웃
- 드랍 아이템
- 아이템 타입
- 아이템 등급
- 국가명
- WorldCountry
- WorldTerritory
- DA_ClientDesignDataAsset
- Zoom Level
- 지역 아이콘
- 지역명
- 추천 레벨
- 셀레탄
- 스켈라

## 숫자/상수/공식
- 텔레포트 거리 기준: 50m 이내
- 줌 배율 판정 기준: 2.5 (2단계와 3단계 중간값)
- 최대 줌 단계: 4단계
- 아이템 정렬 1차: 장비 > 재료 > 소모품
- 아이템 정렬 2차: 전설 > 영웅 > 희귀 > 고급 > 일반
- 아이템 정렬 3차: 아이템 ID 오름차순

## 참조 시스템
- DA_ClientDesignDataAsset
- 월드 및 지역 표시 시트 - ③ 월드 표시 항목

## 주요 섹션
- 기존 기획서 작성 스펙
- ① 임의지역 자동 이동 기능
- ② 미발견 지역 표시
- 월드맵 개선
- ① 월드맵 확대/축소 조작감 개선
- ② 지역 정보 내 드랍 아이템 정렬 순서 변경
- ③ 월드맵 축소 시 국가명 표시
- ④ 월드맵 자동 이동 시 표시 대응
- ⑤ 월드맵 비율 별 아이콘 메시지 비율 조정
- ⑥ 신규 지역 추가 대응
