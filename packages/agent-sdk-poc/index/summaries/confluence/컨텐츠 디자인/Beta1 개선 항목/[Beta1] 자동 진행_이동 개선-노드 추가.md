## 한 줄 설명
자동 진행 및 자동 이동 시 캐릭터가 벽을 비비며 이동하거나 멈추는 현상을 개선하기 위해 노드 기반 경로 시스템을 도입하는 기획.

## 핵심 용어
- 자동 진행
- 자동 이동
- 노드
- 패스 볼륨
- 볼륨 연결
- 내비 매쉬
- 최단 직선 거리
- 길 찾기
- 점 이동
- 텔레포트
- 경로 탐색
- 이동 위치

## 숫자/상수/공식
- 15M (노드 탐색 조건 기준 거리: 목표가 직선 거리 15M 이상일 때 노드 탐색, 15M 이내일 때 직선 이동)

## 참조 시스템
- Path Volume 관련 추가된 기능 (개발 가이드 페이지)

## 주요 섹션
- 방향
- 이슈
- 개선 목표
- 길 만들기
- 배치
- 동작
