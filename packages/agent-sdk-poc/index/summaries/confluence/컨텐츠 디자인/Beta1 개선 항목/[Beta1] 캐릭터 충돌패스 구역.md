## 한 줄 설명
특정 구역 진입 시 플레이어 캐릭터 간 컬리젼 충돌을 비활성화하여 밀집 지역에서의 불편함을 해소하고 서버 부하를 감소시키는 기능.

## 핵심 용어
- 캐릭터 충돌패스 구역
- PlayerCollisionPass
- WorldTerritory 테이블
- 컬리젼 충돌
- 플레이어 캐릭터
- 충돌 무시 기능
- 충돌 비활성화 지역
- 비전투 지역 (safe)
- 컬리젼 대응 스킬
- 끌어오기 스킬
- 길막힘
- 시야 방해
- 충돌 연산

## 숫자/상수/공식
- PlayerCollisionPass 디폴트값: FALSE
- TRUE: 플레이어 간 충돌처리하지 않음
- FALSE: 충돌처리 함
