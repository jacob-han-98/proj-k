## 한 줄 설명
주변 탐색 기능의 연출을 액터 기반에서 FX 기반으로 통일하고, 신규 FX_Scan을 제작하여 시각적 일관성과 리소스 효율성을 개선하는 개선 항목.

## 숫자/상수/공식
- 주변 탐색 범위: 1500 (= 15m)
- FX 크기: 3000x3000
- 최대 퍼져나간 기준 반지름: 15m
