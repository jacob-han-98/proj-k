## 한 줄 설명
Project K의 월드 맵 구성에서 필드, 도시, 마을, 사냥터, 거점의 크기 기준과 이동 동선 폭을 정의하는 레벨 디자인 정책.

- 네임드 몬스터 사냥터
- 필드 보스 사냥터
- 중형 거점
- 소형 거점
- 이동 동선
- 버퍼
- 바리울-기원의 곶

## 숫자/상수/공식
- 필드 최대 사이즈: 500M
- 필드 버퍼: 각 방향 최대 50M
- 도시 크기: 150~200M
- 마을 크기: 100~150M
- 일반 몬스터 사냥터: 30M~35M
- 네임드 몬스터 사냥터: 30M
- 필드 보스 사냥터: 40M
- 중형 거점: 25M
- 소형 거점: 18M
- 사냥터 간 이동 동선 폭: 6M
- 사냥터 내 이동 동선 폭: 7M
- 도시 내 이동 동선 폭: 11M
