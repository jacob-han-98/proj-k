# 몬스터 이동 가이드 (요약)

> 원본: packages/confluence-downloader/output/컨텐츠 디자인/몬스터/몬스터 이동 가이드/content.md

## 한 줄 설명
MonsterClass, MonsterSpawn, ContentSetting에서 참조하는 컬럼을 통해 몬스터의 로밍, 탐색, 추적 범위 및 이동 동작을 정의하는 가이드.

## 핵심 용어
- MonsterClass
- MonsterSpawn
- ContentSetting
- MaxRoamingDistance
- MaxEnemySearchRange
- MaxChaseTime
- MoveRange
- SearchRange
- MonsterMaxRoamStepDistance
- Spawn Volume
- 몬스터 로밍
- 몬스터 탐색
- 적 추적
- 적 추적 거리
- 로밍 범위
- 탐색 범위
- 추적 범위

## 숫자/상수/공식
- 몬스터 적 추적 거리 = MaxRoamingDistance + MaxEnemySearchRange (MonsterClass 기준)
- 몬스터 적 추적 거리 = MoveRange + SearchRange (MonsterSpawn 기준)
- 한 번에 로밍하는 거리 = ContentSetting의 MonsterMaxRoamStepDistance 값 (공통 적용)
