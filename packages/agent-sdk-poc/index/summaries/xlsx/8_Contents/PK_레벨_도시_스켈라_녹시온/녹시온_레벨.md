## 한 줄 설명
Project K의 스켈라 국가 내 도시 레벨 "녹시온"의 지형, 구조, 동선, 오브젝트 배치를 정의하는 레벨 디자인 문서.

## 핵심 용어
- 녹시온
- 스켈라
- 서대륙
- 페리안느 성
- 귀족 대저택
- 광장
- 상점가
- 민가
- 성벽
- 수로
- 정원
- 필드 (물이 마른 땅, 검은 물길, 돌림병 발원지, 안개의 경계, 가시나무숲, 죽음의 은신처)
- 동선
- 텔레포트
- 하수구
- 오염수
- 계단
- 도로 폭
- 높낮이 단차
- 이동 불가 지역
- 플레이 동선

## 숫자/상수/공식
- 전체 사용 면적: 160m × 160m
- 광장 직경: 약 30m
- 도로 폭: 12~14m
- 성벽 높이: 10m
- 성벽 폭: 2m
- 수로 폭: 10m (다리 길이)
- 오브젝트 기본 크기: 10m × 10m (일부 5m × 10m)
- 정원수 높이: 2m, 폭: 3m
- 높이 계층: 4개 층 (민가 < 상점가 < 광장 < 귀족주택가/성)

- ① 위치 (대륙 위치, 레벨 위치, 스켈라 진행 순서, 인접 필드 간 높이 비교)
- ② 크기 (전체 크기, 사용 면적)
- ③ 구조 및 동선 (개략적 구조, 구성 요소, 도로 폭, 구역 나눔, 구역별 설명, 컨셉, 랜드스케이프 참고 이미지 및 높낮이 정보, 플레이 동선, 이동 불가 지역, NPC 위치)
