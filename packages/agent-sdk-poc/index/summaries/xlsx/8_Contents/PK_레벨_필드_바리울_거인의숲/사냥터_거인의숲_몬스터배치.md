- 구울 (도끼, 독침)
- 토우 (되살아난, 거대)

## 숫자/상수/공식
- 구역 번호: 1~5
- 난이도 단계: 5단계 (1=낮음 → 5=높음)
- 몬스터 ID 범위: 10401~10444
- 풀링 ID 예시: 10104101, 10104121, 10104131
- 배치 몬스터 총 20종 (일반 19 + 보스 1)
