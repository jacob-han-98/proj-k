## 한 줄 설명
셀레탄 지역의 필드 사냥터 "배반자들의 소굴"의 배경 설정, 몬스터, NPC, 오브젝트를 정의하는 컨셉 문서. 바리울 유민과 셀레탄 유랑민이 모여 사는 빈민가가 라자루 진리회의 습격으로 파괴되는 스토리를 중심으로 구성.
