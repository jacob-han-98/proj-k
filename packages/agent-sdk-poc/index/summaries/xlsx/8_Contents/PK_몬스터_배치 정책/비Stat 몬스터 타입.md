## 한 줄 설명
Project K의 몬스터를 사거리, 공격 유형, 적극성 3가지 기준으로 8개 타입으로 분류하고, 게임 진행 단계별 필드 내 몬스터 타입 조합 및 배치 비중을 정의하는 문서.

## 핵심 용어
- 비Stat 몬스터 타입
- 사거리 (근거리/원거리)
- 공격 유형 (물리/마법)
- 적극성 (선공/후공)
- 근거리 물리 선공
- 근거리 물리 후공
- 원거리 물리 선공
- 원거리 물리 후공
- 근거리 마법 선공
- 근거리 마법 후공
- 원거리 마법 선공
- 원거리 마법 후공
- MonsterSet
- 필드 포지션
- 극초반/초반/초중반/중반/중반 이후
- 물이 할퀸 땅
- Tier
- Named 등급

## 숫자/상수/공식
- 8개 몬스터 타입 분류
- 극초반: 근거리:원거리 = 80:20, 물리:마법 = 100:0, 후공:선공 = 100:0
- 초반: 근거리:원거리 = 80:20, 물리:마법 = 80:20, 후공:선공 = 100:0
- 초중반: 근거리:원거리 = 70:30, 물리:마법 = 70:30, 후공:선공 = 75:25
- 중반: 근거리:원거리 = 60:40, 물리:마법 = 60:40, 후공:선공 = 50:50
- 중반 이후: 근거리:원거리 = 50:50, 물리:마법 = 50:50, 후공:선공 = 0:100
- 물이 할퀸 땅 예시: 제작 가능 몬스터 수량 18개
