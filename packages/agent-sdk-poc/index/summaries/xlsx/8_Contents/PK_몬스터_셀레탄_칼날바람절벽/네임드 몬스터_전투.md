## 숫자/상수/공식
- 예상 스폰 주기: 5분~30분
- 예상 전투 인원: 1~4명
- 예상 전투 시간: 60초 이하
- 예상 플레이 타임: 1분 내외 (적정 스펙 캐릭터 3명 기준)
- 실린더 크기: 120cm (네임드 몬스터) / 90cm (캐릭터)
- 사이즈 비율: 플레이어 1.5배
- 평타 휘두르기: 피해 대상수 1
- 평타 찌르기: 피해 대상수 1
- 스킬 연속 찌르기: 피해 대상수 2, 쿨타임 3초
- 스킬 2연속 휘두르기: 피해 대상수 5, 쿨타임 5초
- 스킬 천둥 내려찍기: 피해 대상수 5, 쿨타임 5초
