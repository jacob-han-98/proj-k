## 숫자/상수/공식
- 최대 침공 인원: 50명 (외부 서버 인력)
- 침공 구성: 1종 거대 보스 + 최대 50인 외부 서버 PC + 무작위 침 서버 PC
- 밀수 상인 배치: 각 필드당 최소 1기
- 경비병 배치: 2인 1조 (구역 좌우)
- PlayerKill 목표 타입: 10 (불특정 PlayerKill, 서버 구분 X)
- OtherServerPlayerKill 목표 타입: 11 (타 서버 PlayerKill만 카운트)
- 포션 판매 가격: 100~5,000 Gold (아이템별 상이)
