# 참고 (요약)

> 원본: packages/xlsx-extractor/output/8_Contents/PK_스킬 및 버프 아이콘_요청서/참고/_final/content.md

## 한 줄 설명
Project K의 아이템 시스템에서 장비 부위별 역할, 재질 구분, 그리고 게임 내 모든 주요 용어의 한글명·영문명·등급·클래스·스킬 타입 대응표를 정의한 참고 자료.

## 핵심 용어
- 장비 아이템 (EQUIP)
- 비 장비 아이템 (NONE EQUIP)
- 화폐 아이템 (CURRENCY)
- 무기 (SWORD, GREAT SWORD, BOW, STAFF)
- 방어구 (HELMET, JACKET, GLOVES, PANTS, SHOES, SHIRT, CLOAK)
- 액세서리 (EARRING, NECKLACE, RING, BELT)
- 소모성 (POTION, COOK, ENCHANT, BOX)
- 재질 (판금, 가죽, 천)
- 등급 (Epic, Myth, Legendary, Unique, Rare, Uncommon, Common)
- 클래스 (Guardian, Warrior, Archer, Arbalester, Magician, Shaman)
- 스킬 타입 (Active, Passive, Ultimate)
- 메인 공격 스탯
- 메인 방어 스탯
- 서브 방어 스탯
- 특수 스탯

## 숫자/상수/공식
- 재질 3종류: 판금, 가죽, 천
- 무기 종류 4개: 한손검, 양손검, 활, 지팡이
- 방어구 부위 7개: 투구, 셔츠, 상의, 장갑, 하의, 신발, 망토
- 액세서리 부위 4개: 귀걸이, 벨트, 반지, 목걸이
- 소모성 종류 4개: 물약, 요리, 강화 주문서, 상자
- 화폐 종류 2개: 다이아몬드, 골드
- 클래스 6개: 가디언, 워리어, 아처, 아발리스터, 매지션, 샤먼
- 등급 7개: 에픽, 신화, 전설, 영웅, 희귀, 고급, 일반
- 스킬 타입 3개: 액티브, 패시브, 궁극기
