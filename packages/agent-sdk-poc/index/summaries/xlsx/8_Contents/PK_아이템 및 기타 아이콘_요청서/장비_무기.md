## 한 줄 설명
Project K의 무기 아이템 목록을 관리하는 테이블로, 초보자·숙련자·자루판 등급별 6가지 무기 타입(검/방패, 대검, 활, 석궁, 지팡이, 오브)의 데이터명, 제작 설명, 완료 상태를 기록한다.

## 핵심 용어
- 무기 리스트
- 검/방패 (SwordShield)
- 대검 (GreatSword)
- 활 (Bow)
- 석궁 (CrossBow)
- 지팡이 (Staff)
- 오브 (Orb)
- 초보자 등급
- 숙련자 등급
- 자루판 등급
- 데이터명
- 제작 설명
- 완료 여부
- 미기입완료
- 무기 타입 패턴

## 숫자/상수/공식
- 무기 타입 6종 (각 등급별 반복)
- 초보자 등급: 행 1~6
- 숙련자 등급: 행 7~12
- 자루판 등급: 행 13~14 (이후 계속)
- 총 14개 행 (표시됨)
