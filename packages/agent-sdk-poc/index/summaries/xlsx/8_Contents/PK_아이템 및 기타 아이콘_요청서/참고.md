## 숫자/상수/공식
- 재질 종류: 3종류 (판금, 가죽, 천)
- 등급 종류: 7종류 (에픽, 신화, 전설, 영웅, 희귀, 고급, 일반)
- 클래스 종류: 3종류 (검사, 궁사, 마법사)
- 무기 종류: 4종류 (한손검, 양손검, 활, 지팡이)
- 방어구 부위: 7종류 (투구, 상의, 장갑, 하의, 신발, 셔츠, 망토)
- 액세서리 부위: 4종류 (귀걸이, 목걸이, 반지, 벨트)
- 소모성 아이템: 4종류 (물약, 요리, 강화 주문서, 상자)
- 화폐 종류: 2종류 (다이아몬드, 골드)
