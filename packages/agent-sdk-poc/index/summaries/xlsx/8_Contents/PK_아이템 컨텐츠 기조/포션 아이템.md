## 한 줄 설명
게임 내 가장 높은 빈도로 사용되는 회복형 소모품 아이템으로, 포션 전용 UI와 자동 사용 기능을 지원하는 아이템의 정의 및 특징.

## 핵심 용어
- 포션
- HP 회복 포션
- MP 회복 포션
- 버프 포션
- 포션 슬롯
- 포션 전용 UI
- 자동 사용 기능
- 물약 자동 사용 시스템
- 소형 HP 포션
- 중형 HP 포션
- 대형 HP 포션
- 소형 MP 포션
- 중형 MP 포션
- 대형 MP 포션
- 일반 등급
- PC(플레이어 캐릭터)
- 퀵슬롯
- 스택
- Con 스탯

## 숫자/상수/공식
- 최대 스택수: 9,999
- HP 포션 유형: 3종 (소형/중형/대형)
- MP 포션 유형: 3종 (소형/중형/대형, 차후 추가 결정 시)
- 포션 소지량: Con 스탯의 영향을 받음

## 참조 시스템
- 물약 자동 사용 시스템 (별도 문서)

- 기본 사항
- 유형별 특징
- 포션과 등급
