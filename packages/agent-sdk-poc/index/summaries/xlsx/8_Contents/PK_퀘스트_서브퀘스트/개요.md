## 숫자/상수/공식
- 퀘스트 ID 구조: 8자리 (8 + 1 + 1 + 01 + 1 + 01)
- 퀘스트 Volume ID 구조: 4자리 (8 + 01 + 1 + 0001)
- 퀘스트 타입: 1(Main), 2(Sub), 3(Daily), 4(Weekly)
- 클래스: 0(구분없음), 1(검사), 2(궁사), 3(마법사)
- 액트: 1~99
- 챕터: 1~9
- 퀘스트 구분: 1~999
- 볼륨 구분: 1~999999
- 지역: 1(오스트하펜+오염된 농장), 2(남부 해안가), 3(칼날 바람 절벽)
- 퀘스트목표 타입: 0(퀘스트목표), 1(Npc자동 이동), 2(몬스터 이동), 4(트리거볼륨)
- Dialog ID 접미사: +0(수락), +1(목표Arg1), +2(목표Arg2), +3(목표Arg3), +4(완료)
- Beta1 총 퀘스트 수량: 약 243개 (불확실)
