## 한 줄 설명
변신 시스템의 강화 기능에 대한 UI 플로우, 화면 구성, 버튼 동작, 텍스트 키를 정의하는 문서. 안전 강화와 확률 강화 두 가지 강화 방식의 진입, 재료 등록, 결과 연출을 상세히 명시.

## 핵심 용어
- 변신 강화
- 안전 강화
- 확률 강화
- 강화 대상
- 재료 등록
- 자동 등록
- 강화 단계
- 경험치 게이지
- 성공 확률
- 재료 카드
- 등급 필터
- 신규 개방
- 강화 성공
- 강화 실패
- 결과 연출
- 스탯 상승
- 초기화 버튼
- 강화 버튼
- 메타모프 (MetamorphWarning, MetamorphGradeInfo, MetamorphEnchant)
- TextKey

## 숫자/상수/공식
- 안전 강화 최대 단계: 도달 시 경험치 초과분 버림 처리
- 확률 강화 재료 등록: 최소 1장, 최대 4장
- 성공 확률 공식: MetamorphGradeInfo 시트의 EnchantProb 컬럼값 + MetamorphEnchant 시트의 ProbAdjust 컬럼값 (음수 가능, 합산 < 0 시 0 적용)
- 경험치 표시: 소수점 둘째 자리까지 (현재량 및 상승량 모두)
- 100% 초과 경험치: 색상 구분하여 강조 표시
- 200% 초과 시: 100% 초과 색상과 동일
