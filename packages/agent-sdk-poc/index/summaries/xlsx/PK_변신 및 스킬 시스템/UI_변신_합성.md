# UI_변신_합성 (요약)

> 출처: PK_변신 및 스킬 시스템 / UI_변신_합성
> 원본: packages/xlsx-extractor/output/PK_변신 및 스킬 시스템/UI_변신_합성/_final/content.md

## 한 줄 설명
변신 메뉴의 합성 탭에서 재료 카드를 등록하여 새로운 변신 카드를 합성하는 UI 및 플로우를 정의한 문서.

## 핵심 용어
- 합성
- 변신 카드
- 재료 카드
- 합성 포인트
- 전설/영웅/희귀 (등급)
- 자동 등록
- 합성 버튼
- 초기화 버튼
- 결과 대기 화면
- 최종 결과 화면
- 연출 생략
- 합성 실패
- 보유 목록
- 카드 필터
- 합성 가능 횟수

## 숫자/상수/공식
- 합성 가능 횟수: 최소 1회 ~ 최대 11회
- 재료 슬롯: 4개
- 합성 포인트 등급: 전설, 영웅, 희귀 (각각 게이지 표시)

## 참조 시스템
- UI_변신_기본 (결과 대기 화면A, 결과 대기 화면B 참조)
- 변신 시트 - 획득 연출 규칙 및 전체 흐름
- 변신IC54 (합성 시스템 관련 설명)
- 변신IC129 (합성과 강화 차이)

## 주요 섹션
- 기본 정보
- 합성 화면
- 기본 화면
- 자동 등록 설정 UI 정보
- 재료 등록 화면
- 결과 대기 화면A
- 결과 대기 화면B
- 최종 결과 화면
- 유저 플레이 예상 시나리오
