## 숫자/상수/공식
- MP 소모: 30
- 대상 대기 시간: 10초
- 범위: 타겟 주변 5m
- 지속 시간: 5초
- 모든 공격력 +10
- 모든 방어력 +5
- 골드 표시: 90,999
- EXP: 993,999 / 999,999
- 캐릭터 초상화 배열: 2행 4열 (8개)
- 스킬 리스트 페이지 인디케이터: ●○○○
- 슬롯: Slot1~Slot8 (8개)
- MaxCount 입력값 1 = "단일", 2 이상 = "광역"
- Invoke = "발동 역선", ActiveSkill = "액티브 스킬"
