## 한 줄 설명
캐릭터가 몬스터 또는 NPC로 변신하여 고유한 평타, 발동 액션, 스킬을 사용하는 시스템으로, 획득·합성·강화·탈착의 전체 라이프사이클을 정의한다.

## 숫자/상수/공식
- 최초 습득 시 변신 목록에 활성화 (1종)
- 이후 습득분부터 Stack (최대 1,000개 보유, UI 표기 최대 999개)
- 안전 강화 구간: 한 번에 최대 10개 변신 사용 가능
- 확률 강화: 동일 등급 변신 최소 1개~최대 4개 사용 가능
- 합성: 동일 등급 변신 4장 필요
- 강화 단계별 효과 예시 (+1: 방어력 +2, +2: 방어력 +3 발동액션 확률 +3%, +4: 방어력 +4 공격력 +3, +7: 방어력 +8 공격력 +3 발동액션 확률 +5% 스킬 쿨타임 -1초)
- 등급별 적용 스탯 수: 에픽 7개, 신화 7개, 전설 6개, 영웅 5개, 희귀 4개, 고급 3개, 일반 2개
- 등급별 발동 액션: 에픽~영웅 4개, 희귀~고급 3개, 일반 2개
- 등급별 스킬: 에픽 8개, 신화 7개, 전설 6개, 영웅 5개, 희귀 4개, 고급 3개, 일반 2개
