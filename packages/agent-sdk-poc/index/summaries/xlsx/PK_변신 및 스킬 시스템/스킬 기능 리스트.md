## 한 줄 설명
Project K 모바일 MMORPG의 스킬 시스템에 포함되는 기능들(일반 공격, 반복 공격, 끌어오기, 밀쳐내기, 충전형 공격, 장판, 스킬 연결 등)을 정의하고 관리하는 기획 문서.

## 핵심 용어
- NormalAttack (일반 공격)
- RepeatableAttack (반복 공격)
- OnlyBuff (버프 기능)
- Pull (끌어오기)
- Push (밀쳐내기)
- SkillCategoryEnum
- Composite (스킬 연결)
- DarkAttack (자동 공격)
- PulseAnimation (공격 애니메이션)
- PulseAnimBuff (버프 애니메이션)
- AreaShape / AreaSize / AreaStandard / AreaLocation
- Duration / DotCount / DirCount / MaxCount
- AnimationResource
- HitRate
- SkillGroupEnum
- FutureEastAttack (장판 기능)
- DashAttack (돌진 공격)
- AreaFxResourceScale
- Caster (시전자)
- Target (타겟)
- Hit / Collision (충돌)
- Options (옵션 입력)

## 숫자/상수/공식
- Pull 기본값: 충격파 시간(s), 끌어당기는 거리(20=2m)
- Pull 최고 접근 높이: 1.5(약 5cm)
- Push 거리 기준: 500cm 거리는 0.1(x3)
- 장판 예시 Duration: 20초
- 장판 예시 DotCount: 5
- 장판 예시 DirCount: 5
- 장판 예시 MaxCount: 18
- 장판 예시 AreaSize: 350
- 장판 예시 AreaLocation: 100,100
- 장판 예시 Options: 1000,5000
- AreaFxResourceScale 입력값: 10,000 = 100%
- 타겟 당겨올 거리 예시: 500, 100 → 0.5초만에 100cm 앞까지 이동
- 프레임 규칙: 입력값 = 0이면 장판만 통과 가능
