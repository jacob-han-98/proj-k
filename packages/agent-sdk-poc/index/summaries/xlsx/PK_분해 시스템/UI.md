## 한 줄 설명
분해 시스템의 UI 구성 및 상호작용 방식을 정의하며, 메인 UI, 예상 분해 결과 UI, 최종 분해 결과 UI의 세 가지 화면 구조와 각 요소의 기능을 상세히 기술한다.

## 핵심 용어
- 분해 모드
- 인벤토리 슬롯 그리드
- 등급 필터
- 개별 분해 버튼
- 일괄 분해 버튼
- 분해 목록 출력 공간
- 분해 모드 탭
- 선택된 아이템 수량
- 분해 버튼
- 예상 분해 결과 UI
- 확정 보상
- 확률 보상
- DismantleFix
- DismantleRandom
- 최종 분해 결과 UI
- 분해 성공
- 획득 보상 리스트
- 미선택 상태
- 선택 상태
- 잠금 상태 아이템
- 시스템 메시지

## 숫자/상수/공식
- 확정 보상 리스트 최대 출력: 3.5개
- 획득 보상 리스트 최대 출력: 6.5개
- 등급 필터 기본값: 일반, 고급 선택 상태
- 분해 결과 UI 닫기 불가 시간: 약 2초

## 참조 시스템
- 세부 내용 시트 (시스템 메시지 참조)

- 상황별 아이콘 표현
- 메인 UI 상세 테이블
- 예상 분해 결과 UI
- 예상 분해 결과 UI 상세 테이블
- 최종 분해 결과 UI
- 최종 분해 결과 UI 상세 테이블
