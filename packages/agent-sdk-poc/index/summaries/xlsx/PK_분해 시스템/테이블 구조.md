# 테이블 구조 (요약)

> 출처: PK_분해 시스템 / 테이블 구조
> 원본: packages/xlsx-extractor/output/PK_분해 시스템/테이블 구조/_final/content.md

## 한 줄 설명
분해 시스템에서 사용할 신규 테이블(DismantleReward, DismantleRandomReward)과 기존 테이블 수정 사항을 정의하는 데이터베이스 스키마 문서.

## 핵심 용어
- Dismantle.xlsx
- DismantleReward
- DismantleRandomReward
- 확정 보상
- 랜덤 보상
- RewardType
- RewardTypeEnum
- ItemId
- Keyword
- Id
- Prob
- Amount
- AmountMin
- AmountMax
- ItemEquipClass
- DismantleId
- CanDismantle
- 데이터 연결
- 서버 에러 처리

## 숫자/상수/공식
- Prob 총합: 10,000 (=100%)
- Prob 단일값 상한: 10,000 이상 시 서버 에러 처리
- Amount Default: 1
- AmountMin Default: 1
- AmountMax Default: 1
- Prob Default: 0

## 참조 시스템
- RewardTypeEnum (기존 시스템)

## 주요 섹션
- 신규 테이블 - Dismantle.xlsx
- DismantleReward
- DismantleRandomReward
- 기존 테이블 (컬럼 추가)
- ItemEquipClass
