# agent-sdk-poc · 정성 평가 Runbook

이터레이션별로 대표 질문의 답변 발췌 + 한줄 감상을 남긴다.
정량 점수 금지. 질감 변화·개선 방향을 추적하는 용도.

작성 루틴:
1. 코드/인덱스/프롬프트를 고쳤다면
2. `python tests/compare.py --from-file tests/useful_prompts.md --limit 6` 실행
3. 결과 JSON에서 눈에 띄는 변화만 이 파일에 기록 (모든 질문을 복붙할 필요는 없음)
4. 개선 아이디어를 "다음 할 일" 섹션에 적어둔다
