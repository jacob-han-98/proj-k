### 6. 획득 아이템 정보

#### 1) UI 출력 위치 및 방식
- 캐릭터 우측 경험치&골드 획득 UI 하단부터 출력
- 획득 개수에 따라 하단으로 UI가 추가되는 방식
- 동시 출력 개수는 **최대 3종** + 2개 이상의 정보가 연속적으로 출력될 때 정보 간 출력 간격은 **0.5초**
- ✓ 획득 연출 진행 중 추가로 아이템을 획득하여도 정보 간 출력 시간(0.5초)은 영향 받지 않음

#### 2) 상황별 출력 규칙

![획득 아이템 UI 예시 - "MY NAME" 캐릭터 정보 우측에 EXP 999,999, $ 999,999, 아이템명 3개가 표시된 UI](./images/HUD_전투_detail_r1_fig1.png)

**프로젝트K**

| 슬롯 | 설명 |
|------|------|
| [1번칸] | : 획득 아이템 최초 출력 위치 |
| [2번, 3번칸] | : 추가 획득 아이템이 있을 경우 직전 획득한 아이템 정보를 아래로 한 칸씩 밀어 내리는 방식 |

**예시)오딘**
- 오딘 : 아래 → 위로 밀어 올리는 방식
- PK : 위 → 아래로 밀어 내리는 방식

---

**✓ Case.1) 획득 아이템이 3개 이하인 경우**
: 출력 이후 각각 **3초간 유지**하다 제자리에서 사라짐

**✓ Case.2) 획득 아이템이 4개 이상인 경우**
: 3번칸에 출력 중인 아이템 정보가 3초가 되기 전에 2번칸에서 새로운 아이템 정보가 내려올 경우, 3번칸의 이전 정보는 즉시 사라짐
: 마지막으로 획득한 3개 아이템은 Case.1과 동일하게 각각 3초간 유지하다가 제자리에서 사라짐

**✓ Case.3) 획득 연출이 진행 중일 때 추가로 아이템을 획득한 경우**
: 앞서 획득한 아이템들 연출에 이어서 출력되며 출력 순서상 가장 뒤에 예약되는 방식
: 인벤토리에는 이미 획득한 상태이며, 연출만 추가로 출력되는 방식

#### 3) 기타 규칙
- 해당 UI 연출은 몬스터 처치 시 획득 아이템이 있을 경우에만 해 출력
- 상자 아이템 / 퀘스트 / 우편 보상 등으로 획득하는 보상의 경우, 전용 획득 UI가 출력되므로 해당 UI 로의 추가 출력은 하지 않음

### 7. 각종 액션 UI

#### 1) 발동 액션 및 스킬 정보
- 현재 캐릭터가 사용하는 발동 액션 or 스킬 정보 출력
- 좌 → 우 방향으로 출력 & "해당 아이콘 + 액션명" 출력 후, 2초 간 유지 후 사라짐 (최대 1개까지 출력 가능)
- UI 출력 시간이 지나기 전에 새로운 정보가 들어오면 그 즉시 최신 정보로 덮어쓰기
- ✓ 기존에는 최대 3개까지 출력이었으나 채팅 박스 공간 확보를 위해 최대 **1개까지만** 출력
- UI 예시

![발동 액션/스킬 UI 예시 - "발동 천하무적" 아이콘과 텍스트가 좌→우로 펼쳐지는 연출](./images/HUD_전투_detail_r2_fig1.png)

- 아이콘 + "발동" or "스킬" + 액션명 출력
- 좌→우로 펼쳐지는 연출
- "스킬"일 경우, "발동"보다는 강화된 연출 필요

#### 2) 캐스팅 게이지
- 지연 시전이 가능한 스킬 사용 시 출력
- 해당 버튼을 수동으로 누르고 있는 동안 출력
- 누르고 있지 않고 즉시 사용하는 경우에는 출력 X
- 누르고 있는 동안 사망/스턴 등 전투 불가 상태가 되면 출력 중단과 함께 해당 상태로 전환

#### 3) 인터랙션 게이지
- 채집 등 전투 외의 인터랙션 중 일정 시간이 필요한 경우 출력
- 누르고 있지 않아도 인터랙션을 시도하면 자동으로 잔여 시간이 줄어드는 방식
- 인터랙션 중 사망/스턴 등 전투 불가 상태가 되면 출력 중단과 함께 해당 상태로 전환

#### [캐스팅 게이지&인터랙션 게이지 UI 표현 방식]
- 캐스팅 게이지와 인터랙션 UI 디자인 동일하게 사용
- 게이지 하단에 표시되는 문구만 다르게 출력 (Ex. 채집 중, 캐스팅 중)
- 게이지 UI 중앙에 Max 게이지까지의 남은 시간 표시
- 잔여 시간은 소수점 첫째 자리까지 표시되며 실시간으로 줄어드는 방식

≫ HUD 슬롯 자동 사용 관련 컬럼 추가

**[정의]**
- HUD 슬롯에 등록된 아이템의 자동 사용 여부 제어
- ItemType = Consume 이면서 지속 시간이 있는 BuffId가 연결된 아이템의 자동 사용 기능

**[기능 및 조작]**
- 소모품 아이템에 연결된 BuffId의 Duration이 종료된 경우, 자동으로 사용되는 기능
- 기존 스킬 슬롯과 동일하게 해당 HUD 슬롯을 하단으로 내려 동작시키고 다시 올려서 동작을 중단시킬 수 있음

**[규칙]**
- 동작 조건
  → 해당 소모품의 수량이 1개 이상인 경우

- 미동작 조건 (아래 경우 중 1개 이상 해당 시)
  → 해당 소모품의 수량이 0개인 경우
  → 캐릭터가 마을 지역(Town Zone)에 있을 경우

**[UI 및 연출]**
- 기본 연출 방식은 기존의 스킬 슬롯과 동일
  → 자동 사용이 가능한 아이템은 HUD 슬롯에 올려둘 시, 하단에 화살표 마크 출력
  → 자동 사용 중인 아이템은 슬롯의 위치가 하단으로 약간 내려간 뒤 슬롯 상단에 고유 FX 출력
