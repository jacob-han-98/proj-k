## 🎯 개요

**인간병사_일반_검방_근거리**는 방패로 막고 검을 휘두르는 근거리 전투 방식의 인간형 몬스터입니다. 총 3가지 베리 버전(A, B, C)이 존재하며, **국가별 색상 및 배치 지역**에서 차이가 있습니다.

## 🎨 시각적 차이

### 색상 구분

```
기본형 (A): 스켈라 국가 색상
베리1 (B): 초록색 계열 (로바르스)
베리2 (C): 노란색/주황색 계열 (소르브)
```

### 참고 이미지 설명

**기본형 (A)**
- 중세 기사 스타일
- 회색 갑옷
- 둥근 방패 (주황색 바탕에 십자 문양)
- 검 들고 서있는 포즈

**베리1 (B) - 로바르스**
- 녹색 배경 엠블럼
- 흰색 톱니바퀴/기어 문양
- 중앙에 나무 형태 심볼

**베리2 (C) - 소르브**
- 주황색 배경 엠블럼
- 황금색 나무 문양
- 나무 형태가 더 정교하게 표현됨

1. **기능적 차이 없음**: 세 베리 모두 동일한 전투 방식과 능력치를 가짐
2. **시각적 차이**: 국가별 색상 테마에 맞춘 외형 변화

3. **배치 지역 차이**: 각 베리는 특정 국가의 특정 지역에만 배치
4. **세계관 통합**: 동일한 병사 유형이 각 국가의 정체성을 반영하여 변형된 형태
