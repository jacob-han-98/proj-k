# 인간병사 기본형과 베리 버전 차이

> **출처**: PK_몬스터설정 / 인간병사

---

## 1. 시스템 개요

인간병사는 **기본형(A)**, **베리1(B)**, **베리2(C)** 총 3가지 버전으로 구분되며, 각 버전은 **등장 지역**과 **외형 색상**이 다릅니다.

---

## 2. 인간병사 유형별 비교

### 2-1. 인간병사_일반_검방_근거리

| 구분 | 기본형 (A) | 베리1 (B) | 베리2 (C) |
|------|-----------|-----------|-----------|
| **리소스명** | CeletaneSoldier_Shield_A | CeletaneSoldier_Shield_B | CeletaneSoldier_Shield_C |
| **등장 지역** | 셀레탄: 숭배의 언덕 | 로바르스: 비밀 발굴지 | 소르브: 해오름 언덕 |
| **이름(한글)** | 스켈라: 스켈라 병사 | 로바르스: 왕 호위병사 | 소르브: 국경 수비대 기사 |
| **색상 계열** | 기본 (스켈라: 물이 할퀸 땅) | 로바르스 국가의 **초록색 계열** | 소르브 국가의 **노란색/주황색 계열** |

| **이름(한글)** | 스켈라: 스켈라 병사 | 로바르스: 왕 호위석궁병 | 소르브: 국경 수비대 석궁병 |
| **색상 계열** | 기본 (스켈라: 물이 할퀸 땅) | 로바르스 국가의 **초록색 계열** | 소르브 국가의 **노란색/주황색 계열** |

## 3. 공통 특징 (모든 버전 동일)

| 항목 | 내용 |
|------|------|
| **외형 분류** | 인간형 |
| **종족** | 케렌족(서대륙) |
| **외형 특징** | 사슬 갑옷에 투구를 쓰고 있음 |
| **전투 특징 (검방)** | 방패로 막고 검을 휘두르는 형태 |
| **전투 특징 (석궁)** | 석궁으로 쏘는 형태 |
| **등급** | 보급 |

---

## 4. 핵심 차이점 요약

| 차이 요소 | 설명 |
|-----------|------|
| **색상** | 기본형 → 베리1(초록색) → 베리2(노란색/주황색)로 국가별 색상 적용 |
| **등장 지역** | 셀레탄 → 로바르스 → 소르브 지역으로 분산 배치 |
| **명칭** | 각 국가에 맞는 병사 명칭 사용 (스켈라 병사, 왕 호위병사, 국경 수비대 등) |

베리 버전은 **동일한 몬스터를 다른 지역에서 재활용**하면서 **국가별 색상 테마**를 적용하여 시각적 다양성을 확보하는 방식입니다.
