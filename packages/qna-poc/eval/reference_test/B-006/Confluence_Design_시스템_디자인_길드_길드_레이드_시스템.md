# 1.개요

- ****길드 레이드란?****

  - 길드 전용 인스턴트 던전에 진입하여 보스를 처치하고 보상을 받는 컨텐츠

## 1.1 방향성

- ****많은 길드원이 참여할수록 혜택 증가****

  - 보상은 레이드 단계에 따라 등급 증가
  - 참여 인원에 따라 보상 풀 확장 및 확률 업, 버프 획득

## 2.1 레이드 오픈 방법

- 메뉴>길드>길드 정보 화면에서 길드 레이드 버튼 터치

  - 단, 길드 레벨 4 달성 시 활성화된다.

    - 길드 레벨 조건 미달성 상태에 진입 시도 시 시스템 메시지 출력

      - “길드 레벨 4에 개방됩니다.”
- 길드 레이드 창에서 오픈 버튼 터치

  - 단, 아래 조건을 모두 만족해야하며 조건 불만족 시 [시스템 메시지](https://bighitcorp.atlassian.net/wiki/spaces/PK/pages/4898816019#%EC%8B%9C%EC%8A%A4%ED%85%9C-%EB%A9%94%EC%8B%9C%EC%A7%80)를 출력한다.

    - 직책이 길드장/관리자일경우

      - “권한이 없습니다”
    - 입장 횟수 1회 이상 보유

      - “입장 횟수가 부족합니다.”
    - 재화(GuildFund) 보유

      - “길드 자금이 부족합니다”
    - 단계 잠금 해제

      - “개방되지 않는 단계입니다”
    - 길드 레이드가 진행 중이 아닐 때

      - “아직 길드 레이드가 진행 중입니다,”

      - “입장 횟수가 부족합니다”
    - 입장 횟수는 기타 컨텐츠(길드 의뢰)를 통해 추가 획득 가능

## 2.3 제한 시간

- 기존 보스 레이드와 동일한 규칙으로 제한 시간 내에 보스를 처치하지 못하면 클리어 실패

  - 단, 이때 입장권은 차감하지 않음(별도의 어뷰징 요소가 없으므로)
- 제한 시간 밸런스 기준

  - 권장 스펙의 인원이 최소 20명이상 참가(10분 소요)

    - 1인 당 권장 DPS 400, 20명이 10분 동안 피해를 입힌다고 가정
    - 감당할 HP/방어력/회피 스탯 책정
    - 예시) 권장 DPS에 맞춰 방어력과 회피를 책정해서 DPS를 50정도 맞춤으로 50 * 6000 * 20 = 6000000
- 필요 전투력(공명방회)이 높아지고 패턴이 추가되지만 가치가 높은 아이템의 드롭율 높아지는 구조
- 단계별 권장 레벨/공방명회(예시)

## 2.5 예정 시간 공지

- 지정한 요일과 시간을 업데이트하여 모든 길드원에게 표시하는 기능
- 레이드 오픈 권한이 있는 길드원(길드장/관리자)에 한해서만 예정 시간 설정 가능

  - 권한이 없는 길드원은 설정 버튼 숨김 처리
  - 권한이 있다가 사라질 경우 시도 시

    - “권한이 없습니다”
- 기본값은 시간 설정이 안되어 “설정 필요”로 표시

### 2.5.1 설정 방법

- 기본값

  - 월요일 0시00분
- 요일 설정

  - 콤보 박스를 통해 월/화/수/목/금/토/일 요일을 선택할 수 있다.
- 시 설정

  - 슬라이더를 좌우로 움직여 설정

    - 최소값 0
    - 최대값 23
- 분 설정

  - 슬라이더를 좌우로 움직여 설정

    - 최소값 0
    - 최대값 59
- 공지 사용

  - 기본값 Off

    - On 시 설정된 요일/시간 사용
- 저장

  - 변경사항을 저장한다.

    - 변경사항이 있는 상태에서 취소나 나가기 시 확인 팝업 출력

      - “저장하지 않는 내용이 있습니다. 정말 나가시겠습니까?”

# 4. 길드 순위

> ℹ️
>
> 별도의 보상이 없는 명예 순위
>
> 각 서버의 강함 척도를 간접적으로 체감할 수 있다.(ex. 지금 1등 서버가 5단계에 2분 39초)

- 서버와 통합 서버 길드 순위를 표시한다.

  - 순위 / 길드명 / 서버명(통합 서버만) / 클리어 단계 / 클리어 시간
- 우선순위

  - 1순위 - 클리어 단계
  - 2순위 - 클리어 시간

    - 클리어 시간이 동일하다면

      - 기록을 달성한 시간이 빠를수록 우선순위가 높다.

### ContentSetting

- ****GuildRaidCountMax****

  - 길드 레이드 입장 횟수 최대값
  - 주간(WeeklyRestTime)마다 초기화

### GuildRaid

- 길드 레이드에 대해 정의한 테이블
- ****ID****

  - 길드 레이드 식별자
- ****GuildRaidStep****

  - 단계
- ****GuildRaidRecommandMember****

  - 권장 참가 인원
- ****GuildRaidRecommandLevel****

  - 권장 레벨
- ****GuildRaidRecommandCombatPoint****

  - 권장 전투력
- ****GuildRaidUnlockStep****

  - 해당 단계의 잠금 해제를 하기 위한 필요 클리어 레이드 단계
  - 0이면 조건 없음
- ****MonsterID****

  - 보스 몬스터 ID
- ****IconResource****

  - 탭에 들어갈 아이콘 리소스
- ****WorldId****

  - 보스 몬스터 스폰할 레벨
- ****GuildRaidStudio****

  - 길드 레이드 스튜디오 어셋명

### ****GuildRaidReward****

- 길드 레이드 단계별, 인원별 보상을 구분짓는 변수
- ****ID****

  - 구분자
- ****GuildRaidID****

  - 길드 레이드 ID 입력
- ****GuildRaidMemberMin****

  - 최소 인원 수
- ****GuildRaidMemberMax****

  - 최대 인원 수
- ****GuildRaidRewardGuildCoin****

  - 길드 코인
- ****GuildRaidRewardGuildContribution****

  - 주간 기여도
- ****GuildRaidRewardSelfID****

  - 개인 참여 보상
- ****GuildRaidRewardID****

  - 길드 보상 상자 ID
- ****GuildRaidRewardDice****

  - 보상 상자를 굴릴 횟수

### ****GuildRaidBuff****

- 길드 레이드 인원별 버프에 정의한 테이블

  - 해당 버프는 길드 레이드가 종료되면 제거되어야 함
- ****ID****

  - 구분자
- ****GuildRaidMember****

  - 길드 레이드 인원 수
  - 버프 스택에 반영 필요
- ****BuffID****

  - 연결될 버프 ID 입력

    - Array 타입, 복수 개 연결 가능
