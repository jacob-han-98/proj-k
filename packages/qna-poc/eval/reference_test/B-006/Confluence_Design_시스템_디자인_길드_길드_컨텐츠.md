# 방향성

- 길드의 자산화

  - 길드에 시간과 돈을 투자한 만큼 이익을 얻을 수 있도록 함
  - 길드와 개인의 이득이 서로 이어지도록 함
- 다수의 길드가 이득이 되는 구조로 디자인

  - 인원 수가 많아야 유리한 컨텐츠

    - 저투력 인원들도 역할이 있도록 함

# 1. 길드 출석

- 보상 종류

  - 일일 출석 보상

    - 길드 창에 진입 시 체크
    - 일일 길드 보상 지급
  - 전날 출석 보상

    - 전날에 출석한 누적 길드원 수에 따라 차등 보상 지급
    - 0명이면 보상 미지급
  - 단, 신입 길드원의 경우 가입 후 24시간이 지나야 일일/전날 출석 보상 획득 가능
- 갱신 주기

  - 매일 오전 5시 (임시)

    - 만약 로그인을 유지한 상태에서 갱신 시간이 경과할 경우 길드창을 다시 진입해야 갱신
    - 시간은 변경될 여지가 있음
- 보상 지급 시점

  - 별도의 버튼을 누르지 않고 길드 창에 진입만 해도 보상 획득

    - 길드 HUD에 진입한 행위 자체가 사실상 출석이기도 하고, 편의적인 부분을 위한 선택

- 길드 레벨에 따른 컨텐츠 개방 방식

  - 인원 증가

# 3. 길드 기부

- 길드 정보 창에서 길드 기부 버튼을 눌러서 팝업 창 출력
- 기부 종류 마다 기부횟수, 소모되는 재화의 타입과 수량을 다르게 설정 가능
- 기부 종류 마다 일정 확률에 의해 길드 개인 재화 보너스 배율이 적용될 수 있음

  - ContentSetting에서 기본값 설정 가능

    - 길드 레벨의 의해 기부 횟수 증가
    - 보너스 배율은 각 타입별로 지정
- 재화가 부족한 경우 버튼 터치 시 메시지 출력

  - “재화가 부족합니다”
- 보상은 최대 4개까지 가능

## 5.3 세부 규칙

- 자신을 포함한 모든 길드원에게 보상이 지급된다.
- 보상은 라이프 타임이 존재하며 시간이 만료되면 제거된다.
- 수령 규칙

  - 일반 보상은 개별/일괄 수령이 가능
  - 희귀 보상은 개별 수령만 가능
- 일반/희귀 보상 획득 시

  - 선물 레벨 포인트와 특별 상자 열쇠 포인트 획득
- 특별 상자

  - 일반/희귀 상자를 개방하면 얻는 특별 상자 열쇠 포인트로 게이지를 달성하면 획득
  - 개수가 누적될 수 있다.
- 예외처리

  - 선물레벨에 의해 상자 타입이 달라질 경우 처리

    - 순서대로 누적되며 상자마다 개별 만료시간이 적용된다.

      - ex) 초록색 상자 1개 남은 시간 12시간, 이후 파란색 상자 2개 획득(남은 시간 23:59:59)

        - UI상으론 3개가 누적, 초록색 상자와 남은 시간 12시간 표시
        - 초록색 상자 개방 후 파란색 상자와 남은 시간 23:59:XX 표시
    - 만약 특별 상자 게이지가 1이상 올라간 상태에서 선물 레벨이 변경되어 상자 타입이 변경될 경우

      - 특별 상자 게이지 포인트가 인계된다.

        - 초록색 상자 열쇠 포인트 5000 → 파란색 상자로 변경 → 파란색 상자 열쇠 포인트 5000 보유
- 선물 레벨

  - 선물 레벨에 따라 특별 상자의 종류/아이템 구성/세부 확률이 변경된다.

    - 선물의 최대 레벨은 없으며, 무한하게 상승
    - 일정 등급 이상부터 신화 급으로 통일
  - 레벨 별 상자 타입

    - 상자 타입의 등급이 높을 수록, 더 좋은 보상을 지급

|  | ****레벨**** | ****상자 타입**** |
| --- | --- | --- |
| 1 | 1 ~ 10 | 회색 (일반) |
| 2 | 11 ~ 30 | 녹색 (고급) |
| 3 | 31 ~ 60 | 파란색(희귀) |
| 4 | 61 ~ 100 | 보라색(영웅) |
| 5 | 101 ~ 300 | 노란색(전설) |
