# 1.개요

- 인벤토리에서 설정된 규칙에 따라 아이템이 자동 정렬되는 기능

# 2. 규칙

## 2.1 우선순위(기본)

| ****우선순위**** | ****항목**** | ****정렬 기준**** |
| --- | --- | --- |
| 1 | 장착한 장비 | 타입별 우선순위를 따른다 |
| 2 | 타입 | <https://bighitcorp.atlassian.net/wiki/spaces/PK/pages/4928536613#2.2-%ED%83%80%EC%9E%85%EB%B3%84-%EC%9A%B0%EC%84%A0%EC%88%9C%EC%9C%84> 를 따른다  숫자가 낮을수록 우선순위가 높다. |
| 3 | 등급 | 아래 숫자가 높을수록 우선순위가 높다  //main/ProjectK/Resource/design/Enum/GradeEnum.xlsx |
| 4 | 티어 | 숫자가 높을수록 우선순위가 높다 |
| 5 | 인덱스 | 낮을수록 우선순위가 높다 |
| 6 | 획득 시각 | 오래될 수록 우선순위가 높다 |

## 2.2 타입별 우선순위

| ****우선순위**** | ****설명**** |
| --- | --- |
| 1 | ItemTypeEnum의 인덱스 순서를 따른다.   - Equip 1 장비 아이템 - Consume 2 소모성 아이템 - Material 3 재료 아이템 - Quest 4 퀘스트 아이템 - Dummy 5 클라이언트 표기용 아이템 |
| 2 | 장비의 경우에는 아래 순서를 따른다.  신규 컬럼 추가 필요 - EquipPartsEnum의 Arg1 |

## 2.3 정렬 방식

- 정렬 방식의 우선순위 방식을 변경하여 정렬한다.

  - 장착 중인 장비는 어떤 정렬 방식에도 1순위 정렬

| ****정렬 방식**** | ****설명**** | ****우선순위**** |
| --- | --- | --- |
| 타입 | 기본 우선순위 | 1. 장착한 장비 2. 타입 3. 등급 4. 티어 5. 인덱스 6. 획득 시각 |
| 등급 | 등급 높은 우선순위로 정렬 | 1. 장착한 장비 2. 등급 3. 티어 4. 타입 5. 인덱스 6. 획득 시각 |
| 획득 | 인벤토리에 들어온 시각이 오래된 순서대로 정렬한다.  가장 최근에 획득한 아이템이 가장 마지막에 정렬 | 1. 장착한 장비 2. 획득 시각 3. 타입 4. 등급 5. 티어 6. 인덱스 |

## 2.3 정렬 갱신 시점

- 정렬 버튼을 눌렀을 때
- 아이템을 장착했을 때
- ~~아이템을 획득했을 때~~
- 신규 아이템 획득 시 해당 아이템은 최하순위로 순차적으로 정렬
- 인벤토리를 닫을 때

  - 인벤토리가 열려있을 땐 정렬버튼을 누르지 않는 한 자동 정렬 되지 않는다.

## 2.4 정렬 쿨타임

- 정렬 시 부하가 발생한다면 일정 시간동안 정렬이 불가능하도록 처리한다.

  - 시간은 ContentSetting에서 설정

    - 3000 => 3sec
  - 쿨타임 중에는 버튼 딤드 및 스킬 쿨타임과 유사한 UX로 남은 시간 표현

    - 쿨타임 중 정렬 시도 시 시스템 메시지 출력

      - “정렬 재사용 대기 시간이 남아 있습니다.”
